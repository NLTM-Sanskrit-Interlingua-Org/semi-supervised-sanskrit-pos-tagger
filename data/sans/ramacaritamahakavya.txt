atha mālyavataḥ prasthe kākutsthasya viyoginaḥ /
durnivārāśrusaṃvegaḥ jagāma jaladāgamaḥ //
śaśāma vṛṣṭiḥ meghānām utsaṅge tasya bhūbhṛtaḥ /
virarāma na rāmasya dhārāsantatiḥ aśruṇaḥ //
itaḥ tataḥ pariṇatim bheje barhiṇakūjitam /
hā priye rājaputri iti na rāmaparidevitam //
kā api abhikhyā virajasoḥ sūryācandramasoḥ abhūt /
dāśarathyoḥ tathā eva āsīt ayogopahatā ruciḥ //
nirmalendu nabhaḥ reje vikacābjam babhau saraḥ /
param paryaśruvadanau mamlaturbhrātarāvubhau //
smartavyakalikājālam jajñe kuṭajakānanam /
āsīt tathā eva tu manaḥ rāmasya utkalikākulam //
prapede punar udbhedaḥ śucinā kacchaketakaiḥ /
upakriyāyāḥ sadṛśam na ārebhe ravisūnunā //
ākṛṣṭamālatīgandhaḥ siṣeve rāghavam marut /
ānītamaithilīvārtaḥ tam ārtam na tu mārutiḥ //
rāmāya pūrāpagamavyaktoddeśāḥ cakāśire /
āste asmāsu na sītā iti śaṃsantyaḥ iva nimnagāḥ //
yadārdravirahaḥ rāmaḥ prāpitaḥ prāṇasaṃśayam /
nūnam tena anutāpena jagmuḥ jaladarājayaḥ //
dyauḥ muktameghāvaraṇā savitarkonmukham muhuḥ /
yayāce sārasarutaiḥ vicāram iva rāghavam //
diśaḥ dadṛśire tena stenam tam parimārgatā /
mārgadānāparādhinyaḥ gatāḥ dūram bhayāt iva //
tasya ākṛtiviśeṣasya viśeṣāt kaṣṭam īdṛśam /
kacchāḥ kāśacchalāt ūhuḥ iti iva palitāgamam //
paśyantyāḥ iva kaṣṭām tām avasthām maithilīpateḥ /
śuṣkapaṅkacayavyājāt vidadre hṛdayam bhuvaḥ //
vṛṣṭivicchedajātena tānavena asamāptinā /
prapedire jaladharāḥ tasya sabrahmacāritām //
yayau virahasaṃvartaḥ padminīrājahaṃsayoḥ /
apāraduḥkhārṇavayoḥ na sītārāmabhadrayoḥ //
śāntanirjharajhātkāradhārāḥ śikhariṇaḥ babhuḥ /
rāmasya viśadālāpaśuśrūṣānibhṛtāḥ iva //
tasya cakruḥ camatkāram vyatītasamayāḥ api /
smitābhamukulodbhedāḥ kadambavanarājayaḥ //
adṛśyanta puraḥ tena khelāḥ khañjanapaṅktayaḥ /
asmaryanta ca niśvasya priyānayanavibhramāḥ //
abravīt atha kālajñaḥ tatkālapratipattaye /
praṇamya lakṣmaṇaḥ rāmam rāmāvirahaniḥsaham //
jātam vibhātam amalam paśya prāvṛṇṇiśā gatā /
utsāhakamalasya ayam vikāsāvasaraḥ tava //
ucchrayam tīrataravaḥ raveḥ ūrjam abhīṣavaḥ /
madam gopatayaḥ gṛhṇanti abhiyogam jigīṣavaḥ //
prasīda dīyatām karṇaḥ kūjadbhiḥ kuraraiḥ amī /
prārthayanti iva saṃcāram taṭāḥ kardamadurgatāḥ //
bhuvi bhāsvadabhīṣūṇām tvadiṣūṇām ca śatruṣu /
ayam sampatitum kālaḥ sa duṣkālaḥ ativāhitaḥ //
sudūronnataparyantāḥ tīryante saṃkramaiḥ iva /
analpagādhāḥ saritaḥ tarubhiḥ pūrapātitaiḥ //
āviṣkṛtaviśuddhibhyām nadyaḥ sumahilāḥ iva /
kulābhyām iva kūlābhyām āpatsu api cakāsati //
jalajānām sumanasām vayasām jalasevinām /
asya pratijalādhāram sāmagryam avalokyate //
yuyutsārabhasotkhātasaritkhātakarodhasaḥ /
kukudmantaḥ madakalāḥ kalayanti jaganti adhaḥ //
viśveśa kuru viśvāsam āvartante na te ghanāḥ /
amī śyāmalayanti āśāḥ śukāḥ śālivanonmukhāḥ //
mṛgāḥ samprati śāleyam sampannam sampatanti amī /
sāketam adhitiṣṭhantam arthinaḥ tvām iva unmukhāḥ //
kāryadik gṛhyatām kā api kim ārya sthīyate vṛthā /
evam eva ayam asmākam iṣṭaḥ api kim upaiṣyati //
vijayāya bhavāsīnaḥ mayi arpitavapurbharaḥ /
mūrcchānuvalanavyastaḥ prastaraḥ tyajyatām ayam //
pratīṣṭopatyakāsāladalāgrajalabindavaḥ /
sāryantām kṣaṇam uddhūya sthagitāṃsataṭāḥ jaṭāḥ //
muhūrtam kriyatām ārya kapolavirahī karaḥ /
vyuṣitaḥ valkalagranthiḥ aṃsāt unmocyatām ayam //
itaḥ vitatya dīyantām iṣavaḥ klinnayantraṇāḥ /
nirmuktārdrajaraccailam ādattām āyatam dhanuḥ //
itaḥ avatīryatām prāsthāt asvāsthyam idam ujjhyatām /
jahi iṣubhiḥ daśagrīvam daśadigvedhaśodhitaiḥ //
ṛjuḥ ādīyatām panthāḥ kāryakanthā garīyasī /
utsāryatām idam dūram tamaḥ kiṃkāryatāmayam //
āryāyāḥ śivam eva asti śatruḥ nihataḥ eva te /
alam dattvā avasādāya trailokyaśaraṇam vapuḥ //
rākṣasāpaśadaḥ kaḥ asau tvayi sajyaśarāsane /
anena te avasādena dūram āropitaḥ punaḥ //
aśruvegaiḥ pramīlābhiḥ vilāpaiḥ akhilaḥ sa te /
āmṛṣṭadehayātrasya māsaḥ prauṣṭhapadaḥ gataḥ //
kriyatām adhunā utsāhaḥ cintyatām nidhanam dviṣaḥ /
kaḥ svabāhusahāyānām mahatām ātmanigrahaḥ //
bhadram bhādrapadacchedacchinnābhrājinarajjave /
tubhyam puruṣanāgāya nirvighnatarase adhunā //
prasīda kuru bhūyaḥ api dehayātrām yathā tathā /
anullaṅghitavijñaptiḥ lakṣmaṇaḥ avarajeṣu te //
agādham dhairyam āryasya triṣu lokeṣu gīyate /
janamātrocitaḥ kaḥ ayam ākalpakaparigrahaḥ //
aropaśamitodagrajāmadagnyāgnitejasaḥ /
pratīcchati śarāsāram tava ārya ruṣitasya kaḥ //
svakṛtyabhārasannyāsī yāvat ittham viṣīdasi /
tāvat antakadaṃṣṭrāṇām arātiḥ atithiḥ na te //
ātatāyī sa vijñātaḥ jīvati āryā iti ca śrutam /
prāptaḥ yātrāsahaḥ kālaḥ kālakṣepasya kaḥ guṇaḥ //
na jñāyate param vyaktam areḥ māyāvinaḥ padam /
tanmārgaṇam ca suhṛdā sugrīveṇa pratiśrutam //
diṅmukhāhūtavikhyātakapiyūthaḥ kapīśvaraḥ /
kṣaṇāt nūnam asau āryapādamūle patiṣyati //
śālipiṅgamahīpṛṣṭhaspardhayā iva bhaviṣyati /
raṃhasviharisaṃghātasampātakapilam nabhaḥ //
teṣām āroṣaruciraiḥ acirāt mukhamaṇḍalaiḥ /
drakṣyate taruṇādityaśatacchannam iva ambaram //
aho dīptiḥ aho kāntiḥ aho śīlam aho balam /
aho śaktiḥ aho bhaktiḥ aho prajñā hanūmataḥ //
pragalbhena apramattena tena abhyantaramantriṇā /
nūnam asmatkṛte svaptum sugrīvāya na dīyate //
sa ca pratyupakārāya tvarayati enam ādṛtaḥ /
dhruvam buddhivayovṛddhaḥ sacivaḥ jāmbavān api //
tat evam hetunā kena vānarendraḥ vilambate /
na asau kṛtaghnasya bhavet guṇavadbhṛtyasaṃgrahaḥ //
āgantavyam dhruvam tena mārgitā āryā pravṛttinā /
prapannaparakṛtyānām atyuṣṇaḥ hi samudyamaḥ //
tvatprasādāptasambhogasukhasaṃmīlitena vā /
na tena amūḥ dadṛśire diśaḥ proṣitakālikāḥ //
kiyacciram punar asau tasya na śrutim eṣyati /
krīḍātaṭākahaṃsānām ninādaḥ madamūrchitaḥ //
uktaiḥ kim atha bhūyobhiḥ adya śvaḥ vā āgamiṣyati /
paryutsukitakiṣkindhaḥ satyasandhaḥ harīśvaraḥ //
vyatireke api yat kāryam tadapyārya pradhāryatām /
gṛhyatām kāryasiddhiḥ naḥ tadvilambatiraskṛtā //
yaḥ tathā upakṛtaḥ pūrvam āryeṇa āryākṛte adhunā /
sa śītalaḥ syāt athavā paracittāni vetti kaḥ //
sa pratiśrutam artham naḥ sphītaḥ vismṛtavān iva /
gatvā āśu smāryatām ārya kāryataptasya kā kṣamā //
tam aham bodhayiṣyāmi daṇḍagarbhābhiḥ uktibhiḥ /
sāma saṃmīlayati eva sukhasaktān pramādinaḥ //
prasīda jīyatām mohaḥ kaḥ aham iti avadhīyatām /
drutam uttīrṇaḥ eva āste vyasanābdhiḥ kiyān ayam //
sūnṛtaiḥ iti saumitreḥ prītaḥ pīyūṣavarṣibhiḥ /
tam antarvedanāvegam jigāya iva raghūdvahaḥ //
unmīlayāmāsa dṛśau diśaḥ kiṃcit alokata /
unnanāma ca talpāntāt mocayan pakṣmalāḥ jaṭāḥ //
satvaraprahvasaumitricīrotsāritareṇuni /
śucini kvacit āsanne niṣasāda śilātale //
upaviṣṭaḥ tadā rāmaḥ na reje sītayā vinā /
dhvajaḥ nirvaijayantīkaḥ paurandaraḥ iva ucchritaḥ //
anūpaviṣṭam anujam danujārim iva adrijit /
pratyabhāṣata vātsalyagadgadāni padāni saḥ //
ehi ehi priyasaṃlāpa hlādayā āliṅgitena mām /
ruddham ruddham upaiti aśru tvām api īkṣe na lakṣmaṇa //
hā vatsa bhavataḥ api iyam avasthā aṅgeṣu vartate /
dhriyatām rāmahatakaḥ kleśayan ekabāndhavam //
na asti vatsa cikitsā asya vyādheḥ vidhurajanmanaḥ /
yasya yāpanayā kṣīṇam idam te sāntvanāmṛtam //
naikadurvāraghorādhighaṭṭanācalitāḥ api /
tava āśvāsanayā vatsa na gacchanti mama asavaḥ //
pituḥ naḥ priyamitrasya lūnapakṣasya maddviṣā /
saṃsmaran na utsahe vatsa marmacchedam jaṭāyuṣaḥ //
tātasya tātamitrasya vatsavatsalayoḥ tayoḥ /
aham eva gataḥ pāpaḥ nidhanāya nimittatām //
priyāpahārajaḥ śāmyet samyak unmūlite ripau /
manyuḥ dehāvadhiḥ ayam tātamitravipattibhūḥ //
nabhastalāt khagapatau patati cchinnapakṣatau /
kaḥ aparaḥ tām mama suhṛt mā bhaiṣīḥ iti abhāṣata //
syāt vatsa rāmahatakāt klībaḥ kaḥ puruṣaḥ aparaḥ /
ekapatnīm anugatām arāteḥ yaḥ na rakṣati //
tayā teṣu tapasvinyā sīdatoḥ āvayoḥ api /
daṇḍakāraṇyakaṣṭeṣu mukham na vidhurīkṛtam //
videharājasya sutā snuṣā daśarathasya sā /
hatarāmasya jāyā iti bhramati sma vane vane //
svam api apuṣṇatā kāyam api jāyām arakṣatā /
mayā samaḥ śrutaḥ kaḥ api jātaḥ klībaḥ raghoḥ kule //
tvam kevalam na subhrātā tathā vilapatā muhuḥ /
tiryañcaḥ api mayā amuṣmin vane lakṣmaṇa roditāḥ //
dikṣu sannaddhameghāsu mama ghorādhisākṣiṇām /
avahan nirjharanibhān nagānām aśrunimnagāḥ //
ārāt ākarṇitasnigdhanavābdaninadaiḥ api /
matkāruṇyāt araṇye api na anṛtyata kalāpibhiḥ //
satyam eṣyati sugrīvaḥ satyam jīvati jānakī /
tvadvacaḥ na viparyeti bhrātar brūhi punaḥ punaḥ //
tvadvākyaśīkaraiḥ ebhiḥ niruddhabahirudgamaḥ /
mama antar leḍhi marmāṇi sītāvirahamurmuraḥ //
tasmāt api dahati uccair ayam mām aparaḥ śikhī /
yat vanaukasi nirvaire muktaḥ vālini mārgaṇaḥ //
gurvī punaḥ ca lajjā iyam uttamarṇaiḥ iva adhunā /
mārgitavyaḥ yat asmābhiḥ plavagāḥ pratyupakriyām //
na asti pratyupakārāśā tatra naḥ capale kapau /
yena atilaghunā uttuṅgaḥ laṅghitaḥ satyapādapaḥ //
uttiṣṭha vatsa gacchāvaḥ sādhayāmaḥ anyataḥ adhunā /
navaiśvaryasukhavyagraḥ sugrīvaḥ nāgam iṣyati //
tapasvī ramatām evam cirāt dāraiḥ samāgataḥ /
sa pīḍyamānaḥ praṇayāt virasaḥ kim kariṣyati //
kim ca vatsa daśāsāmyāt jātāḥ smaḥ suhṛdaḥ purā /
tasya adhunā tu vistīrṇavibhavāndhasya ke vayam //
gṛhītasādhuvartmā api sadyaḥ samprāpya sampadam /
patati skhalitaḥ atīva pramattaḥ prahate pathi //
sa vatsa viralaḥ jantuḥ vipatkālapratiśrutam /
sadyaḥ sampanmadaḥ yasya na vilumpati cetasi //
astu sādhuḥ asādhuḥ vā na vicāryaḥ sa me adhunā /
vatsa na utsahate cetaḥ tatra dogdhum upakriyām //
asau anāgataḥ śreyān āgacchan vatsa vāryatām /
praveśayanti suhṛdam na dhīrāḥ svārthasaṃkaṭe //
kartavyā upakṛtiḥ sadbhiḥ yatra tatra vipadgate /
avimṛśya kṛtaḥ asmābhiḥ sa punaḥ kṛpaṇaḥ paṇaḥ //
kim kṣiptam acalaprāntāt antaḥsuptaśivāśatam /
nirdeśapuruṣeṇa iva kaṅkālam dundubheḥ mayā //
kasya arthasya kṛte viddhāḥ sapta sālāḥ tapasvinaḥ /
sa mā pratyetu vaidheyaḥ kim na sādhitam anyataḥ //
akṣātram kṣatriyayuvā kaḥ nu rāmaḥ iva kṣipet /
viśikham vālini vyagre sugrīvasya paśoḥ kṛte //
na parityajyate mārgaḥ tāvat nirvācyakardamaḥ /
yāvat pihitakaṣṭāya kāryāya ātmā na dīyate //
iṣvā ekayā mayā viddhaḥ bāliśena valīmukhaḥ /
anuddhārāḥ śarāḥ tena ropitāḥ mayi vāṅmayāḥ //
anavāptavipatpāraḥ smariṣyāmi kiyanti vā /
vatsa kālocitam śādhi na rāmaḥ kiṃcit īkṣate //
avalambasva mām bhrātaḥ sītāsmaraṇanissaham /
dūram punaḥ prabhavatā manaḥ mohena lupyate //
sanāthobhayapārśvasya tvayā vatsa tayā ca me /
vaneṣu ca sataḥ kaṣṭam ajaniṣṭa na kiṃcana //
hā bhīru hā priyatame hā madvirahakātare /
kva asi iti ardhoktinissaṃjñaḥ jagāma sa mahītalam //
mātaḥ kaikeyi tuṣṭā asi bhuṅkṣva rājyam akaṇṭakam /
vadan iti apatat tasya lakṣmaṇaḥ caraṇābjayoḥ //
tataḥ tau sāśrunayanau mṛgaiḥ vidhuravīkṣitau /
nipetatuḥ divaḥ iva bhraṣṭau vidhuravī kṣitau //
iti nipatitayoḥ tayoḥ dvayoḥ daśarathanandanayoḥ mahītale /
daśabhiḥ api digaṅganāmukhaiḥ adhṛtivaśāt iva tatyaje ruciḥ //
ete nikāmarasikasya jayanti pādāḥ śrīhāravarṣayuvarājamahītalendoḥ /
yaiḥ dvādaśārkakiraṇotkaradurnivāraḥ sṛṣṭaḥ abhinandakumudasya mahāvikāsaḥ //
iti abhinandakṛtau rāmacarite mahākāvye prathamaḥ sargaḥ samāptaḥ //
tām anīkṣitacarīm adhīratām ātmanaḥ atha viniyamya lakṣmaṇaḥ /
sāntvanārtham iti vācam abravīt tīvramanyurayamūḍham agrajam //
kāryam ārya pariśiṣṭam ucyatām mucyatām idam ayogajam tamaḥ /
rājatu udyamadivākarodayasmeram āśu mukhapaṅkajam tava //
āsitum samayaḥ eṣa na āvayoḥ moham ittham avalambitum na ca /
utsahasva nanu hṛṣyatām cirādiṣu adhīnam adhunā abhikāṅkṣitam //
kevalam phalabhujaḥ na tasya naḥ taskarasya nilaye asti niścayaḥ /
abhyupaitu ravinandanaḥ kṣaṇāt andhakāram idam api apaiṣyati //
aurasān api sutān arīn iva ghnanti kāraṇavaśāt mahībhujaḥ /
nigrahāt anujavidviṣaḥ kapeḥ kim pṛthagjanaḥ iva anutapyase //
krīḍatām mṛgavaneṣu dhanvinām jīvakoṭikadane api kā kṣatiḥ /
khidyase kim aniśam prayojanāt ekavānaravadhaḥ kutaḥ kṛtaḥ //
tasya ca pravayasaḥ jaṭāyuṣaḥ svargiṇaḥ kim adhunā anuśocyate /
yena jarjarakalevaravyayāt krītam indukiraṇojjvalam yaśaḥ //
etat ekam ativāhyatām ahaḥ śvaḥ na sa plavagarāṭ pratīkṣyate /
tat prasīda paritaḥ vicīyatā ātmanā eva sanagārṇavā avaniḥ //
tat kabandhaśabarīhanūmatām āvayoḥ ajani saṅgatam yathā /
iṣṭaśaṃsi vanaśailapānthayoḥ kim tathā anyat api na āpatet punaḥ //
yatra yatra tava siddhiḥ agrataḥ kaḥ kariṣyati na te samīhitam /
ujjvale kṛtavidām na kevalam tena vartmani durātmanā sthitam //
sambhavati abhimaraḥ pure athavā tasya tīvram ayaśaḥ kim ucyate /
duḥkham utkhanati kaṇṭakāvalīḥ aṅgasandhiṣu navaḥ kila īśvaraḥ //
samprati prathamam ārya gṛhyatām tatpravṛttim upalabdhum udyamaḥ /
marṣaṇīyaparilambadūṣaṇaḥ saḥ adhunā api akapaṭaḥ ghaṭate cet //
muñca moham avatīryatām itaḥ sevyatām iyam anāvilā nadī /
eṣa gacchati javāt japāruṇaḥ vāruṇīm aruṇasārathiḥ diśam //
unmimīla raghunandanaḥ cirāt tat niśamya vacanam camatkṛtaḥ /
api ayogavidhurāḥ na śerate saṃmukhīnavidhayaḥ tathāvidhāḥ //
muktamoham avalokya lakṣmaṇaḥ tam mṛgāṅkam iva meghanirgatam /
ādade atha śaśikāntabhūdharasyandabinduśiśirām sarasvatīm //
eṣa paśya patitaḥ nabhastalāt astaśailagahaneṣu naśyati /
sandhyayā caramadikpragalbhayā bhagnasāndrakarapañjaraḥ khagaḥ //
śakyam arcitum anuccapāṇibhiḥ pāśapāṇinagarīnivāsibhiḥ /
sānurodhaḥ iva dūrabandhuraḥ bandhujīvakusumāruṇaḥ raviḥ //
eṣa muktakarapāṭavaḥ raviḥ nāṭayati anuguṇatvam appatau /
āpadi prakṛtiḥ ujjhitā varam na āśrayasya visadṛk viceṣṭitam //
bimbamūlarucirāḥ raveḥ amī recayanti gaganam gabhastayaḥ /
pūrayanti ca mahīm mahīruhām āyatāḥ pratiśarīrayaṣṭayaḥ //
agrataḥ śikharaśākhisaṃhateḥ chāyayā upacitayā apasāritaḥ /
pṛṣṭhataḥ atinibhṛtam mahībhṛtām ātapaḥ kaṭakam āśu na ujjhati //
svīkaroti śanakaiḥ anokahacchāyam adriśikharasthalīḥ api /
hīyate nijavapuṣi api kramāt ātapasya mamatā tapasvinaḥ //
appateḥ diśi viceṣṭate raviḥ vyākulāḥ śakunayaḥ svananti amī /
dhvāntadṛṣṭiviṣadarśanāt dine durnivāranidhane nimīlati //
kīryate kṛtaravaiḥ khagavrajaiḥ eṣa tallataṭacaityapādapaḥ /
sāyam adhvanikaṭe nirargalaḥ sajjanālayaḥ iva pravāsibhiḥ //

na prāpnuvanti kṣaṇam āptavācaḥ sadādṛtāḥ cāraṇacāṭuvādāḥ //
vadanti bhītāḥ sacivāḥ na kiṃcit kim cintayā asmadgatayā viṭānām /
svayam na samprati anusaṃdadhāti pūrvāparam kāmaparaḥ sa rājā //

itthaṃgate tatra gatasya kena manyuḥ tava udīrṇarayaḥ nivāryaḥ /
vicāyamāṇaḥ na guṇāya naḥ syāt asau kapīnām kadanānubandhaḥ //
prapannaśāṭhyasya haṭhopanāmāt āneṣyamāṇasya guṇam na vīkṣe /
tathāvidhāḥ hi dviṣatām stavena kaṣanti karṇau param antikasthāḥ //
spṛṣ.aḥ na cet vikriyayā kayā api na upasthitaḥ kāryavaśāt idānīm /
etu svayam na kriyate vṛthā eva sa tvadbalātkāraniruddhaśobhaḥ //
yathā āvayoḥ tātajaṭāyuṣā uktam uktam kabandhena yathā ca tena /
tathā upayāsyati acirāt avaśyam āhūtacakraḥ haricakravartī //
yat satyam ādityasamudbhavatvāt udbhūtim eṣyanti guṇāḥ tadīyāḥ /
tulāgrasampātavilambadoṣe tasmin visaṃvādam aham na śaṅke //
viropataptasya videśavāsāt ājagmuṣaḥ tasya nijādhivāsam /
iyanti ahāni utsukabandhuvargasaṃvargaṇavyagratayā eva yānti //
sa nūnam āyāsyati vānarendraḥ saṃdeśakārpaṇyam asaṃmatam me /
sarvasvabhūtam visṛjanti vatsa kāryātipāte api na dhairyam āryāḥ //
vyatītavighnaḥ suhṛdarthanighnaḥ na śītalaḥ sthāsyati kim gatena /
sa tvaryamāṇaḥ tvaramāṇaḥ eva dhairyasya naḥ pāram avāpsyati iva //
dhik astu mām tam prati vipratīpam āśaṅkitam tat bahu yena yat vā /
svakāryadolāsu janaḥ alpabhāgyaḥ prāk niścinoti vyatirekam eva //
ittham na mām pratyupapattim eti tatra ādyapakṣadvitaye api yātrā /
varam visoḍhāni kiyanti api iha saṅkhyātakālābhyadhikāni ahāni //
svālocitam vaktu bhavān idānīm iti anuṣṭheyam asaṃśayam yat /
bālyāt prabhṛti eva tava uttarā dhīḥ na pūrvapakṣe abhiniveśam eti //
iti prasannām giram agrajasya śrutvā cirāt vītaviṣādapaṅkaḥ /
mukhaśriyā eva ujjvalayā abhinandan idam sumitrātanayaḥ jagāda //
samarthitam sādhu samastam eva kṛtī kapiḥ kim bahunā uditena /
pravyaktam api etat adoṣadṛṣṭyā gṛhītam āryeṇa na yat tadāgaḥ //
tasya ūrjitena atiraveḥ avadyam na eva īkṣate antaḥkaraṇam mama api /
ayam punaḥ tena kutaḥ na jānāmi atikramaḥ kālakṛtaḥ na dṛṣṭaḥ //

yat kiṃcit ādiśya nibhena kim mām niṣedhasi kleśam iva īkṣamāṇaḥ //

āryaḥ anurodhāt na yunakti yatra tatra aham ambāniyamāt svatantraḥ //

tam vairisaṃhāramahopakāram āryāya niryātayitum yatena //

anvīkṣate yasya mama avasādāt āryaḥ sahāyāntaram antarajñaḥ //

alakṣitoddeśam api dviṣantam nihatya na āyānti hi yat kṣaṇena //

asṛgbhujām bhoktum asṛk tvarantām bhūyaḥ api bhūcandra tava ardhacandrāḥ //
ātāṭakāsaṃjñapanāt prapannaḥ tvayā eṣa varṇāśramarakṣaṇāya /
samāpyatām śeṣanibarhaṇena kṣapācarāṇām nidhanādhikāraḥ //

āsīt kṣuraprasya kiyān vilambaḥ krūreṣu teṣu triśiraḥśiraḥsu /
kati iṣavaḥ dūṣaṇadehayātrānivāraṇārtham ca raṇe visṛṣṭāḥ //
kasya avalambāt akarot parāsum mārīcam uccāvacamāyam āryaḥ /
jñātā ca dik saṃjñapitaḥ tvayā eva kṣapācarāṇām adhipaḥ ca saḥ api //
iti aprapañcoditakāraṇāni mithaḥ tayoḥ tathyaguṇagrahāṇi /
neduḥ stuvānāḥ iva sūnṛtāni khagāḥ kulāyeṣu kṛtaikamatyāḥ //
āropya sandhyām diśi vajrapāṇeḥ paryasya candram kakubhi pratīcyām /
utsaṅgitoḍuprakarā khagānām ādāya nidrām rajanī jagāma //
kaviḥ ca jīvaḥ ca sutaḥ ca bhūmeḥ sa dakṣiṇāśātilakaḥ muniḥ ca /
āsannasūryāṃśubhayadrutasya kṣaṇam rarakṣuḥ bhagaṇasya pṛṣṭham //
dik darśayāmāsa mukham sarāgam na yāvat aindrī gaṇikā iva mattā /
tāvat dvijeśaḥ aṅkam iyāya sindhoḥ sandhyāsurāśīkarasekaśaṅkī //
visarjitoḍuḥ kṣaṇam induḥ āsīt āsīt upānteṣu api na andhakāram /
bhṛśam diśi indrasya rarāja sandhyā sindūriṇī iva graharājamudrā //
uccetum uccaiḥ uḍukuḍmalāni vyākurvatī rāgam atipragāḍham /
śākhām iva ālambya diśam maghonaḥ sandhyā viyat bhūruham āruroha //
śeṣoḍupuṣpastabakoccayāya saṃjātarāgātiśayā iva sandhyā /
api uccakaiḥ pūrvamahīdhramūrdhni sthitvā viyadbhūruham āruroha //
