śriyaḥ kurūṇām adhipasya pālanīm prajāsu vṛttim yam ayuṅkta veditum /
sa varṇiliṅgī viditaḥ samāyayau yudhiṣṭhiram dvaitavane vanecaraḥ  //
kṛtapraṇāmasya mahīm mahībhuje jitām sapatnena nivedayiṣyataḥ  /
na vivyathe tasya manaḥ na hi priyam pravaktum icchanti mṛṣā hitaiṣiṇaḥ  //
dviṣām vighātāya vidhātum icchataḥ rahasi anujñām adhigamya bhūbhṛtaḥ  /
sa sauṣṭhavaudāryaviśeṣaśālinīm viniścitārthām iti vācam ādadhe //
kriyāsu yuktaiḥ nṛpa cāracakṣuṣaḥ na vañcanīyāḥ prabhavaḥ anujīvibhiḥ  /
ataḥ arhasi kṣantum asādhu sādhu vā hitam manohāri ca durlabham vacaḥ  //
sa kiṃsakhā sādhu na śāsti yaḥ adhipam hitāt na yaḥ saṃśṛṇute sa kiṃprabhuḥ  /
sadā anukūleṣu hi kurvate ratim nṛpeṣu amātyeṣu ca sarvasampadaḥ  //
nisargadurbodham abodhaviklavāḥ kva bhūpatīnām caritam kva jantavaḥ  /
tava anubhāvaḥ ayam abodhi yat mayā nigūḍhatattvam nayavartma vidviṣām //
viśaṅkamānaḥ bhavataḥ parābhavam nṛpāsanasthaḥ api vanādhivāsinaḥ  /
durodaracchadmajitām samīhate nayena jetum jagatīm suyodhanaḥ  //
tathā api jihmaḥ sa bhavajjigīṣayā tanoti śubhram guṇasampadā yaśaḥ  /
samunnayan bhūtim anāryasaṃgamāt varam virodhaḥ api samam mahātmabhiḥ  //
kṛtāriṣaḍvargajayena mānavīm agamyarūpām padavīm prapitsunā /
vibhajya naktaṃdinam astatandriṇā vitanyate tena nayena pauruṣam //
sakhīn iva prītiyujaḥ anujīvinaḥ samānamānān suhṛdaḥ ca bandhubhiḥ  /
sa santatam darśayate gatasmayaḥ kṛtādhipatyām iva sādhu bandhutām //
asaktam ārādhayataḥ yathāyatham vibhajya bhaktyā samapakṣapātayā /
guṇānurāgāt iva sakhyam īyivān na bādhate asya trigaṇaḥ parasparam //
niratyayam sāma na dānavarjitam na bhūri dānam virahayya satkriyām /
pravartate tasya viśeṣaśālinī guṇānurodhena vinā na satkriyā //
vasūni vāñchan na vaśī na manyunā svadharmaḥ iti eva nivṛttakāraṇaḥ  /
gurūpadiṣṭena ripau sute api vā nihanti daṇḍena sa dharmaviplavam //
vidhāya rakṣān paritaḥ paretarān aśaṅkitākāram upaiti śaṅkitaḥ  /
kriyāpavargeṣu anujīvisātkṛtāḥ kṛtajñatām asya vadanti sampadaḥ  //
anāratam tena padeṣu lambhitāḥ vibhajya samyak viniyogasatkriyām /
phalanti upāyāḥ paribṛṃhitāyatīḥ upetya saṃgharṣam iva arthasampadaḥ  //
anekarājanyarathāśvasaṃkulam tadīyam āsthānaniketanājiram /
nayati ayugmacchadagandhiḥ ārdratām bhṛśam nṛpopāyanadantinām madaḥ  //
sukhena labhyāḥ dadhataḥ kṛṣīvalaiḥ akṛṣṭapacyāḥ iva sasyasampadaḥ  /
vitanvati kṣemam adevamātṛkāḥ cirāya tasmin kuravaḥ cakāsati //
mahaujasaḥ mānadhanāḥ dhanārcitāḥ dhanurbhṛtaḥ saṃyati labdhakīrtayaḥ  /
na saṃhatāḥ tasya na bhedavṛttayaḥ priyāṇi vāñchanti asubhiḥ samīhitum  //
udārakīrteḥ udayam dayāvataḥ praśāntabādham diśataḥ abhirakṣayā /
svayam pradugdhe asya guṇaiḥ upasnutā vasūpamānasya vasūni medinī //
mahībhujām saccaritaiḥ caraiḥ kriyāḥ sa veda niḥśesam aśeṣitakriyaḥ  /
mahodayaiḥ tasya hitānubandhibhiḥ pratīyate dhātuḥ iva īhitam phalaiḥ  //
na tena sajyam kvacit udyatam dhanuḥ na vā kṛtam kopavijihmam ānanam /
guṇānurāgeṇa śirobhiḥ uhyate narādhipaiḥ mālyam iva asya śāsanam //
sa yauvarājye navayauvanoddhatam nidhāya duḥśāsanam iddhaśāsanaḥ  /
makheṣu akhinnaḥ anumataḥ purodhasā dhinoti havyena hiraṇyaretasam //
pralīnabhūpālam api sthirāyati praśāsat āvāridhi maṇḍalam bhuvaḥ  /
sa cintayati eva bhiyaḥ tvat eṣyatīḥ aho durantā balavadvirodhitā //
kathāprasaṅgena janaiḥ udāhṛtāt anusmṛtākhaṇḍalasūnuvikramaḥ  /
tava abhidhānāt vyathate natānanaḥ sa duḥsahāt mantrapadāt iva uragaḥ  //
tat āśu kartum tvayi jihmam udyate vidhīyatām tatra vidheyam uttaram /
parapraṇītāni vacāṃsi cinvatām pravṛttisārāḥ khalu mādṛśām giraḥ  //
iti īrayitvā giram āttasatkriye gate atha patyau vanasaṃnivāsinām /
praviśya kṛṣṇā sadanam mahībhujā tat ācacakṣe anujasannidhau vacaḥ  //
niśamya siddhim dviṣatām apākṛtīḥ tataḥ tatastyāḥ viniyantum akṣamā /
nṛpasya manyuvyavasāyadīpinīḥ udājahāra drupadātmajā giraḥ  //
bhavādṛśeṣu pramadājanoditam bhavati adhikṣepaḥ iva anuśāsanam /
tathā api vaktum vyavasāyayanti mām nirastanārīsamayāḥ durādhayaḥ  //
akhaṇḍam ākhaṇḍalatulyadhāmabhiḥ ciram dhṛtā bhūpatibhiḥ svavaṃśajaiḥ  /
tvayā svahastena mahī madacyutā mataṅgajena srak iva apavarjitā //
vrajanti te mūḍhadhiyaḥ parābhavam bhavanti māyāviṣu ye na māyinaḥ  /
praviśya hi ghnanti śaṭhāḥ tathāvidhān asaṃvṛtāṅgān niśitāḥ iva iṣavaḥ  //
guṇānuraktām anuraktasādhanaḥ kulābhimānī kulajām narādhipaḥ  /
paraiḥ tvadanyaḥ kaḥ iva apahārayet manoramām ātmavadhūm iva śriyam //
bhavantam etarhi manasvigarhite vivartamānam naradeva vartmani /
katham na manyuḥ jvalayati udīritaḥ śamītarum śuṣkam iva agniḥ ucchikhaḥ  //
avandhyakopasya nihantuḥ āpadām bhavanti vaśyāḥ svayam eva dehinaḥ  /
amarṣaśūnyena janasya jantunā na jātahārdena na vidviṣā ādaraḥ  //
paribhraman lohitacandanocitaḥ padātiḥ antargiri reṇurūṣitaḥ  /
mahārathaḥ satyadhanasya mānasam dunoti no kaccit ayam vṛkodaraḥ  //
vijitya yaḥ prājyam ayacchat uttarān kurūn akupyam vasu vāsavopamaḥ  /
sa valkavāsāṃsi tava adhunā āharan karoti manyum na katham dhanaṃjayaḥ  //
vanāntaśayyākaṭhinīkṛtākṛtī kacācitau viṣvak iva agajau gajau /
katham tvam etau dhṛtisaṃyamau yamau vilokayan utsahase na bādhitum //
imām aham veda na tāvakīm dhiyam vicitrarūpāḥ khalu cittavṛttayaḥ  /
vicintayantyāḥ bhavadāpadam parām rujanti cetaḥ prasabham mama ādhayaḥ  //
purā adhirūḍhaḥ śayanam mahādhanam vibodhyase yaḥ stutigītimaṅgalaiḥ  /
adabhradarbhām adhiśayya sa sthalīm jahāsi nidrām aśivaiḥ śivārutaiḥ  //
purā upanītam nṛpa rāmaṇīyakam dvijātiśeṣeṇa yat etat andhasā /
tat adya te vanyaphalāśinaḥ param paraiti kārśyam yaśasā samam vapuḥ  //
anāratam yau maṇipīṭhaśāyinau arañjayat rājaśiraḥsrajām rajaḥ  /
niṣīdataḥ tau caraṇau vaneṣu te mṛgadvijālūnaśikheṣu barhiṣām //
dviṣannimittā yat iyam daśā tataḥ samūlam unmūlayati iva me manaḥ  /
paraiḥ aparyāsitavīryasampadām parābhavaḥ api utsavaḥ eva māninām //
vihāya śāntim nṛpa dhāma tat punaḥ prasīda saṃdhehi vadhāya vidviṣām /
vrajanti śatrūn avadhūya niḥspṛhāḥ śamena siddhim munayaḥ na bhūbhṛtaḥ  //
puraḥsarāḥ dhāmavatām yaśodhanāḥ suduḥsaham prāpya nikāram īdṛśam /
bhavādṛśāḥ cet adhikurvate ratim nirāśrayā hanta hatā manasvitā //
atha kṣamām eva nirastavikramaḥ cirāya paryeṣi sukhasya sādhanam /
vihāya lakṣmīpatilakṣma kārmukam jaṭādharaḥ san juhudhi iha pāvakam //
na samayaparirakṣaṇam kṣamam te nikṛtipareṣu pareṣu bhūridhāmnaḥ  /
ariṣu hi vijayārthinaḥ kṣitīśāḥ vidadhati sopadhi saṃdhidūṣaṇāni //
vidhisamayaniyogāt dīptisaṃhārajihmam śithilavasum agādhe magnam āpatpayodhau /
riputimiram udasya udīyamānam dinādau dinakṛtam iva lakṣmīḥ tvām samabhyetu bhūyaḥ  //
vihitām priyayā manaḥpriyām atha niścitya giram garīyasīm /
upapattimat ūrjitāśrayam nṛpam ūce vacanam vṛkodaraḥ  //
yat avocata vīkṣya māninī paritaḥ snehamayena cakṣuṣā /
api vāgadhipasya durvacam vacanam tat vidadhīta vismayam //
viṣamaḥ api vigāhyate nayaḥ kṛtatīrthaḥ payasām iva āśayaḥ  /
sa tu tatra viśeṣadurlabhaḥ sat upanyasyati kṛtyavartma yaḥ  //
pariṇāmasukhe garīyasi vyathake asmin vacasi kṣataujasām /
ativīryavati iva bheṣaje bahuḥ alpīyasi dṛśyate guṇaḥ  //
iyam iṣṭaguṇāya rocatām rucirārthā bhavate api bhāratī /
nanu vaktṛviśeṣaniḥspṛhāḥ guṇagṛhyāḥ vacane vipaścitaḥ  //
catasṛṣu api te vivekinī nṛpa vidyāsu nirūḍhim āgatā /
katham etya matiḥ viparyayam kariṇī paṅkam iva avasīdati //
vidhuram kim ataḥ param paraiḥ avagītām gamite daśām imām /
avasīdati yat suraiḥ api tvayi sambhāvitavṛtti pauruṣam //
dviṣatām udayaḥ sumedhasā guruḥ asvantataraḥ sumarṣaṇaḥ  /
na mahān api bhūtim icchatā phalasampatpravaṇaḥ parikṣayaḥ  //
acireṇa parasya bhūyasīm viparītām vigaṇayya ca ātmanaḥ  /
kṣayayuktim upekṣate kṛtī kurute tatpratikāram anyathā //
anupālayatām udeṣyatīm prabhuśaktim dviṣatām anīhayā /
apayānti acirāt mahībhujām jananirvādabhayāt iva śriyaḥ  //
kṣayayuktam api svabhāvajam dadhatam dhāma śivam samṛddhaye /
praṇamanti anapāyam utthitam pratipaccandram iva prajāḥ nṛpam //
prabhavaḥ khalu kośadaṇḍayoḥ kṛtapañcāṅgavinirṇayaḥ nayaḥ  /
sa vidheyapadeṣu dakṣatām niyatim lokaḥ iva anurudhyate //
abhimānavataḥ manasvinaḥ priyam uccaiḥ padam ārurukṣataḥ  /
vinipātanivartanakṣamam matam ālambanam ātmapauruṣam //
vipadaḥ abhibhavanti avikramam rahayati āpadupetam āyatiḥ  /
niyatā laghutā nirāyateḥ agarīyān na padam nṛpaśriyaḥ  //
tat alam pratipakṣam unnateḥ avalambya vyavasāyavandhyatām /
nivasanti parākramāśrayāḥ na viṣādena samam samṛddhayaḥ  //
atha cet avadhiḥ pratīkṣyate katham āviṣkṛtajihmavṛttinā /
dhṛtarāṣṭrasutena sutyajyāḥ ciram āsvādya narendrasampadaḥ  //
dviṣatā vihitam tvayā athavā yadi labdhā punar ātmanaḥ padam /
jananātha tava anujanmanām kṛtam āviṣkṛtapauruṣaiḥ bhujaiḥ  //
madasiktamukhaiḥ mṛgādhipaḥ karibhiḥ vartayati svayam hataiḥ  /
laghayan khalu tejasā jagat na mahān icchati bhūtim anyataḥ  //
abhimānadhanasya gatvaraiḥ asubhiḥ sthāsnu yaśaḥ cicīṣataḥ  /
acirāṃśuvilāsacañcalā nanu lakṣmīḥ phalam ānuṣaṅgikam //
jvalitam na hiraṇyaretasam cayam āskandati bhasmanām janaḥ  /
abhibhūtibhayāt asūn ataḥ sukham ujjhanti na dhāma māninaḥ  //
kim apekṣya phalam payodharān dhvanataḥ prārthayate mṛgādhipaḥ  /
prakṛtiḥ khalu sā mahīyasaḥ sahate na anyasamunnatim yayā //
kuru tat matim eva vikrame nṛpa nirdhūya tamaḥ pramādajam /
dhruvam etat avehi vidviṣām tvadanutsāhahatāḥ vipattayaḥ  //
dviradān iva digvibhāvitān caturaḥ toyanidhīn iva āyataḥ  /
prasaheta raṇe tava anujān dviṣatām kaḥ śatamanyutejasaḥ  //
jvalataḥ tava jātavedasaḥ satatam vairikṛtasya cetasi /
vidadhātu śamam śivetarā ripunārīnayanāmbusantatiḥ  //
iti darśitavikriyam sutam marutaḥ kopaparītamānasam /
upasāntvayitum mahīpatiḥ dviradam duṣṭam iva upacakrame //
apavarjitaviplave śucay hṛdayagrāhiṇi maṅgalāspade /
vimalā tava vistare girām matiḥ ādarśe iva abhidṛśyate //
sphuṭatā na padaiḥ apākṛtā na ca na svīkṛtam arthagauravam /
racitā pṛthagarthatā girām na ca sāmarthyam apohitam kvacit //
upapattiḥ udāhṛtā balāt anumānena na ca āgamaḥ kṣataḥ  /
idam īdṛk anīdṛgāśayaḥ prasabham vaktum upakrameta kaḥ  //
avitṛptatayā tathā api me hṛdayam nirṇayam eva dhāvati /
avasāyayitum kṣamāḥ sukham na vidheyeṣu viśeṣasampadaḥ  //
sahasā vidadhīta na kriyām avivekaḥ param āpadām padam /
vṛṇate hi vimṛśyakāriṇam guṇalubdhāḥ svayam eva sampadaḥ  //
abhivarṣati yaḥ anupālayan vidhibījāni vivekavāriṇā /
sa sadā phalaśālinīm kriyām śaradam lokaḥ iva adhitiṣṭhati //
śuci bhūṣayati śrutam vapuḥ praśamaḥ tasya bhavati alaṃkriyā /
praśamābharaṇam parākramaḥ sa nayāpāditasiddhibhūṣaṇaḥ  //
matibhedatamastirohite gahane kṛtyavidhau vivekinām /
sukṛtaḥ pariśuddhaḥ āgamaḥ kurute dīpaḥ iva arthadarśanam //
spṛhaṇīyaguṇaiḥ mahātmabhiḥ carite vartmani yacchatām manaḥ  /
vidhihetuḥ ahetuḥ āgasām vinipātaḥ api samaḥ samunnateḥ  //
śivam aupayikam garīyasīm phalaniṣpattim adūṣitāyatim /
vigaṇayya nayanti pauruṣam vijitakrodharayāḥ jigīṣavaḥ  //
apaneyam udetum icchatā timiram roṣamayam dhiyā puraḥ  /
avibhidya niśākṛtam tamaḥ prabhayā na aṃśumatā api udīyate //
balavān api kopajanmanaḥ tamasaḥ na abhibhavam ruṇaddhi yaḥ  /
kṣayapakṣe iva aindavīḥ kalāḥ sakalāḥ hanti sa śaktisampadaḥ  //
samavṛttiḥ upaiti mārdavam samaye yaḥ ca tanoti tigmatām /
adhitiṣṭhati lokam ojasā sa vivasvān iva medinīpatiḥ  //
kva cirāya parigrahaḥ śriyām kva ca duṣṭendriyavājivaśyatā /
śaradabhracalāḥ calendriyaiḥ asurakṣāḥ hi bahucchalāḥ śriyaḥ  //
kim asāmayikam vitanvatā manasaḥ kṣobham upāttaraṃhasaḥ  /
kriyate patiḥ uccakaiḥ apām bhavatā dhīratayā adharīkṛtaḥ  //
śrutam api adhigamya ye ripūn vinayante sma na śarīrajanmanaḥ  /
janayanti acirāya sampadām ayaśaḥ te khalu cāpalāśrayam //
atipātitakālasādhanā svaśarīrendriyavargatāpanī /
janavat na bhavantam akṣamā nayasiddheḥ apanetum arhati //
upakārakam āyateḥ bhṛśam prasavaḥ karmaphalasya bhūriṇaḥ  /
anapāyi nibarhaṇam dviṣām na titikṣāsamam asti sādhanam //
praṇatipravaṇān vihāya naḥ sahajasnehanibaddhacetasaḥ  /
praṇamanti sadā suyodhanam prathame mānabhṛtām na vṛṣṇayaḥ  //
suhṛdaḥ sahajāḥ tathā itare matam eṣām na vilaṅghayanti ye /
vinayāt iva yāpayanti te dhṛtarāṣṭrātmajam ātmasiddhaye //
abhiyogaḥ imān mahībhujaḥ bhavatā tasya tataḥ kṛtāvadheḥ  /
pravighāṭayitā samutpatan haridaśvaḥ kamalākarān iva //
upajāpasahān vilaṅghayan sa vidhātā nṛpatīn madoddhataḥ  /
sahate na janaḥ api adhaḥkriyām kim u lokādhikadhāma rājakam //
asamāpitakṛtyasampadām hatavegam vinayena tāvatā /
prabhavanti abhimānaśālinām madam uttambhayitum vibhūtayaḥ  //
madamānasamuddhatam nṛpam na viyuṅkte niyamena mūḍhatā /
atimūḍhaḥ udasyate nayāt nayahīnāt aparajyate janaḥ  //
aparāgasamīraṇeritaḥ kramaśīrṇākulamūlasantatiḥ  /
sukaraḥ taruvat sahiṣṇunā ripuḥ unmūlayitum mahān api //
aṇuḥ api upahanti vigrahaḥ prabhum antaḥprakṛtiprakopajaḥ  /
akhilam hi hinasti bhūdharam taruśākhāntanigharṣajaḥ analaḥ  //
matimān vinayapramāthinaḥ samupekṣeta samunnatim dviṣaḥ  /
sujayaḥ khalu tādṛk antare vipadantāḥ hi avinītasampadaḥ  //
laghuvṛttitayā bhidām gatam bahiḥ antaḥ ca nṛpasya maṇḍalam /
abhibhūya harati anantaraḥ śithilam kūlam iva apagārayaḥ  //
anuśāsatam iti anākulam nayavartmākulam arjunāgrajam /
svayam arthaḥ iva abhivāñchitaḥ tam abhīyāya parāśarātmajaḥ  //
madhuraiḥ avaśāni lambhayan api tiryañci śamam nirīkṣitaiḥ  /
paritaḥ paṭu bibhrat enasām dahanam dhāma vilokanakṣamam //
sahasā upagataḥ savismayam tapasām sūtiḥ asūtiḥ enasām /
dadṛśe jagatībhujā muniḥ sa vapuṣmān iva puṇyasaṃcayaḥ  //
atha uccakaiḥ āsanataḥ parārdhyāt udyan sa dhūtāruṇavalkalāgraḥ  /
rarāja kīrṇākapiśāṃśujālaḥ śṛṅgāt sumeroḥ iva tigmaraśmiḥ  //
avahitahṛdayaḥ vidhāya saḥ arhām ṛṣivat ṛṣipravare gurūpadiṣṭām /
tadanumatam alaṃcakāra paścāt praśamaḥ iva śrutam āsanam narendraḥ  //
vyaktoditasmitamayūkhavibhāsitoṣṭhaḥ tiṣṭhan muneḥ abhimukham sa vikīrṇadhāmnaḥ  /
tanvantam iddham abhitaḥ gurum aṃśujālam lakṣmīm uvāha sakalasya śaśāṅkamūrteḥ  //
tataḥ śaraccandrakarābhirāmaiḥ utsarpibhiḥ prāṃśum iva aṃśujālaiḥ  /
bibhrāṇam ānīlarucam piśaṅgīḥ jaṭāḥ taḍitvantam iva ambuvāham //
prasādalakṣmīm dadhatam samagrām vapuḥprakarṣeṇa janātigena /
prasahya cetaḥsu samāsajantam asaṃstutānām api bhāvam ārdram //
anuddhatākāratayā viviktām tanvantam antaḥkaraṇasya vṛttim /
mādhuryavisrambhaviśeṣabhājā kṛtopasambhāṣam iva īkṣitena //
dharmātmajaḥ dharmanibandhinīnām prasūtim enaḥpraṇudām śrutīnām /
hetum tadabhyāgamane parīpsuḥ sukhopaviṣṭam munim ābabhāṣe //
anāptapuṇyopacaraiḥ durāpā phalasya nirdhūtarajāḥ savitrī /

adya kriyāḥ kāmadughāḥ kratūnām satyāśiṣaḥ samprati bhūmidevāḥ  /

samujjhitajñātiviyogakhedam tvatsaṃnidhau ucchvasati iva cetaḥ  //

tathā api kalyāṇakarīm giram te mām śrotum icchā mukharīkaroti //
iti uktavān uktiviśeṣaramyam manaḥ samādhāya jayopapattau /
udāracetāḥ giram iti udārām dvaipāyanena abhidadhe narendraḥ  //
cicīṣatām janmavatām alaghvīm yaśovataṃsām ubhayatra bhūtim /
abhyarhitā bandhuṣu tulyarūpā vṛttiḥ viśeṣeṇa tapodhanānām //
tathā api nighnam nṛpa tāvakīnaiḥ prahvīkṛtam me hṛdayam guṇaughaiḥ  /
vītaspṛhāṇām api muktibhājām bhavanti bhavyeṣu hi pakṣapātāḥ  //

yaḥ tyaktavān vaḥ sa vṛthā balāt vā moham vidhatte viṣayābhilāṣaḥ  //
jahātu na enam katham arthasiddhiḥ saṃśayya karṇādiṣu tiṣṭhate yaḥ  /
asādyuyogāḥ hi jayāntarāyāḥ pramāthinīnām vipadām padāni //
pathaḥ cyutāyām samitau ripūṇām dharmyām dadhānena dhuram cirāya /
tvayā vipatsu api avipatti ramyam āviṣkṛtam prema param guṇeṣu //
vidhāya vidhvaṃsanam ātmanīnam śamaikavṛtteḥ bhavataḥ chalena /
prakāśitatvanmatiśīlasārāḥ kṛtopakārāḥ iva vidviṣaḥ te //
labhyā dharitrī tava vikrameṇa jyāyān ca vīryāstrabalaiḥ vipakṣaḥ  /
ataḥ prakarṣāya vidhiḥ vidheyaḥ prakarṣatantrā hi raṇe jayaśrīḥ  //
triḥsaptakṛtvaḥ jagatīpatīnām hantā guruḥ yasya sa jāmadagnyaḥ  /
vīryāvadhūtaḥ sma tadā viveda prakarṣam ādhāravaśam guṇānām //
yasmin anaiśvaryakṛtavyalīkaḥ parābhavam prāptaḥ iva antakaḥ api /
dhunvan dhanuḥ kasya raṇe na kuryāt manaḥ bhayaikapravaṇam sa bhīṣmaḥ  //
sṛjantam ājāviṣusaṃhatīḥ vaḥ saheta kopajvalitam gurum kaḥ  /
parisphurallolaśikhāgrajihvam jagat jighatsantam iva antavahnim //
nirīkṣya saṃrambhanirastadhairyam rādheyam ārādhitajāmadagnyam /
asaṃstuteṣu prasabham bhayeṣu jāyeta mṛtyoḥ api pakṣapātaḥ  //
yayā samāsāditasādhanena suduścarām ācaratā tapasyām /
ete durāpam samavāpya vīryam unmīlitāraḥ kapiketanena //
mahattvayogāya mahāmahimnām ārādhanīm tām nṛpa devatānām /
dātum pradānocita bhūridhāmnīm upāgataḥ siddhim iva asmi vidyām //
iti uktavantam vraja sādhaya iti pramāṇayan vākyam ajātaśatroḥ  /
prasedivāṃsam tam upāsasāda vasan iva ante vinayena jiṣṇuḥ  //
niryāya vidyā tha dinādiramyāt bimbāt iva arkasya mukhāt maharṣeḥ  /
pārthānanam vahnikaṇāvadātā dīptiḥ sphuratpadmam iva abhipede //
yogam ca tam yogyatamāya tasmai tapaḥprabhāvāt vitatāra sadyaḥ  /
yena asya tattveṣu kṛte avabhāse samunmimīla iva cirāya cakṣuḥ  //
ākāram āśaṃsitabhūrilābham dadhānam antaḥkaraṇānurūpam /
niyojayiṣyan vijayodaye tam tapaḥsamādhau muniḥ iti uvāca //
anena yogena vivṛddhatejāḥ nijām parasmai padavīm ayacchan /
samācara ācāram upāttaśastraḥ japopavāsābhiṣavaiḥ munīnām //
kariṣyase yatra suduścarāṇi prasattaye gotrabhidaḥ tapāṃsi /
śiloccayam cāruśiloccayam tam eṣa kṣaṇāt neṣyati guhyakaḥ tvām //
iti bruvāṇena mahendrasūnum maharṣiṇā tena tirobabhūve /
tam rājarājānucaraḥ asya sākṣāt pradeśam ādeśam iva adhitasthau //
kṛtānatiḥ vyāhṛtasāntvavāde jātaspṛhaḥ puṇyajanaḥ sa jiṣṇau /
iyāya sakhyau iva samprasādam viśvāsayati āśu satām hi yogaḥ  //
atha uṣṇabhāsā iva sumerukuñjān vihīyamānān udayāya tena /
bṛhaddyutīn duḥkhakṛtātmalābham tamaḥ śanaiḥ pāṇḍusutān prapede //
asaṃśayālocitakāryanunnaḥ premṇā samānīya vibhajyamānaḥ  /
tulyāt vibhāgāt iva tanmanobhiḥ duḥkhātibhāraḥ api laghuḥ sa mene //
dhairyeṇa viśvāsyatayā maharṣeḥ tīvrāt arātiprabhavāt ca manyoḥ  /
vīryam ca vidvatsu sute maghonaḥ sa teṣu na sthānam avāpa śokaḥ  //
tān bhūridhāmnaḥ caturaḥ api dūram vihāya yāmān iva vāsarasya /
ekaughabhūtam tat aśarma kṛṣṇām vibhāvarīm dhvāntam iva prapede //
tuṣāralekhākulitotpalābhe paryaśruṇī maṅgalabhaṅgabhīruḥ  /
agūḍhabhāvā api vilokane sā na locane mīlayitum viṣehe //
akṛtrimapremarasābhirāmam rāmārpitam dṛṣṭivilobhi dṛṣṭam /
manaḥprasādāñjalinā nikāmam jagrāha pātheyam iva indrasūnuḥ  //
dhairyāvasādena hṛtaprasādā vanyadvipena iva nidāghasindhuḥ  /
niruddhabāṣpodayasannakaṇṭham uvāca kṛcchrāt iti rājaputrī //
magnām dviṣacchadmani paṅkabhūte sambhavānām bhūtim iva uddhariṣyan /
ādhidviṣām ā tapasām prasiddheḥ asmat vinā mā bhṛśam unmanībhūḥ  //
yaśaḥ adhigantum sukhalipsayā vā manuṣyasaṃkhyām ativartitum vā /
nirutsukānām abhiyoggabhājām samutsukā iva aṅkam upaiti siddhiḥ  //
lokam vidhātrā vihitasya goptum kṣattrasya muṣṇan vasu jaitram ojaḥ  /
tejasvitāyāḥ vijayaikavṛtteḥ nighnan priyam prāṇam iva abhimānam //
vrīḍānataiḥ āptajanopanītaḥ saṃśayya kṛcchreṇa nṛpaiḥ prapannaḥ  /
vitānabhūtam vitatam pṛthivyām yaśaḥ samūhan iva digvikīrṇam //
vīryāvadāneṣu kṛtāvamarṣaḥ tanvan abhūtām iva sampratītim /
kurvan prayāmakṣayam āyatīnām arkatviṣām ahnaḥ iva avaśeṣaḥ  //
prasahya yaḥ asmāsu paraiḥ prayuktaḥ smartum na śakyaḥ kim uta adhikartum /
navīkariṣyati upaśuṣyat ārdraḥ sa tvat vinā me hṛdayam nikāraḥ  //
prāptaḥ abhimānavyasanāt asahyam dantī iva dantavyasanāt vikāram /
dviṣatpratāpāntaritorutejāḥ śaradghanākīrṇaḥ iva ādiḥ ahnaḥ  //
savrīḍamandaiḥ iva niṣkriyatvāt nātyartham astraiḥ avabhāsamānaḥ  /
yaśaḥkṣayakṣīṇajalārṇavābhaḥ tvam anyam ākāram iva abhipannaḥ  //
duḥśāsanāmarṣarajovikīrṇaiḥ ebhiḥ vinā arthaiḥ iva bhāgyanāthaiḥ  /
keśaiḥ kadarthīkṛtavīryasāraḥ kaccit saḥ eva asi dhanaṃjayaḥ tvam //
sa kṣattriyaḥ trāṇasahaḥ satām yaḥ tat kārmukam karmasu yasya śaktiḥ  /
vahan dvayīm yadi aphale arthajāte karoti asaṃskārahatām iva uktim //
vītaujasaḥ sannidhimātraśeṣāḥ bhavatkṛtām bhūtim apekṣamāṇāḥ  /
samānaduḥkhāḥ iva naḥ tvadīyāḥ sarūpatām pārtha guṇāḥ bhajante //
ākṣipyamāṇam ripubhiḥ pramādāt nāgaiḥ iva ālūnasaṭam mṛgendram /
tvām dhūḥ iyam yogyatayā adhirūḍhā dīptyā dinaśrīḥ iva tigmaraśmim //
karoti yaḥ aśeṣajanātiriktām sambhāvanām arthavatīm kriyābhiḥ  /
saṃsatsu jāte puruṣādhikāre na pūraṇī tam samupaiti saṃkhyā //
priyeṣu yaiḥ pārtha vinā upapatteḥ vicintyamānaiḥ klamam eti cetaḥ  /
tava prayātasya jayāya teṣām kriyāt aghānām maghavā vighātam //
mā gāḥ cirāya ekacaraḥ pramādam vasan asambādhaśive api deśe /
mātsaryarāgopahatātmanām hi skhalanti sādhuṣu api mānasāni //
tat āśu kurvan vacanam maharṣeḥ manorathān naḥ saphalīkuruṣva /
pratyāgatam tvā asmi kṛtārtham eva stanopapīḍam parirabdhukāmā //
udīritām tām iti yājñasenyā navīkṛtodgrāhitaviprakārām /
āsādya vācam sa bhṛśam didīpe kāṣṭhām udīcīm iva tigmaraśmiḥ  //
atha abhipaśyan iva vidviṣaḥ puraḥ purodhasā āropitahetisaṃhatiḥ  /
babhāra ramyaḥ api vapuḥ sa bhīṣaṇam gataḥ kriyām mantraḥ iva abhicārikīm //
avilaṅghyavikarṣaṇam paraiḥ prathitajyāravakarma kārmukam /
agatau aridṛṣṭigocaram śitanistriṃśayujau maheṣudhī //
yaśasā iva tirodadhat muhuḥ mahasā gotrabhidāyudhakṣatīḥ  /
kavacam ca saratnam udvahan jvalitajyotiḥ iva antaram divaḥ  //
akalādhipabhṛtyadarśitam śivam urvīdharavartma samprayān /
hṛdayāni samāviveśa sa kṣaṇam udbāṣpadṛśām tapobhṛtām //
anujaguḥ atha divyam dundubhidhvānam āśāḥ surakusumanipātaiḥ vyomni lakṣmīḥ vitene /
priyam iva kathayiṣyan āliliṅga sphurantīm bhuvam anibhṛtavelāvīcibāhuḥ payodhiḥ  //
tataḥ sa kūjatkalahaṃsamekhalām sapākasasyāhitapāṇḍutāguṇām /
upāsasāda upajanam janapriyaḥ priyām iva āsāditayauvanām bhuvam //
vinamraśāliprasavaughaśālinīḥ apetapaṅkāḥ sasaroruhāmbhasaḥ  /
nananda paśyan upasīma sa sthalīḥ upāyanībhūtaśaradguṇaśriyaḥ  //
nirīkṣyamāṇāḥ iva vismayākulaiḥ payobhiḥ unmīlitapadmalocanaiḥ  /
hṛtapriyādṛṣṭivilāsavibhramāḥ manaḥ asya jahruḥ śapharīvivṛttayaḥ  //
tutoṣa paśyan kalamasya saḥ adhikam savārije vāriṇi rāmaṇīyakam /
sudurlabhe na arhati kaḥ abhinanditum prakarṣalakṣmīm anurūpasaṃgame //
nunoda tasya sthalapadminīgatam vitarkam āviṣkṛtaphenasaṃtati /
avāptakiñjalkavibhedam uccakaiḥ vivṛttapāṭhīnaparāhatam payaḥ  //
kṛtormirekham śithilatvam āyatā śanaiḥ śanaiḥ śāntarayeṇa vāriṇā /
nirīkṣya reme sa samudrayoṣitām taraṅgitakṣaumavipāṇḍu saikatam //
manoramam prāpitam antaram bhruvoḥ alaṃkṛtam kesarareṇuṇā aṇunā /
alaktatāmrādharapallavaśriyā samānayantīm iva bandhujīvakam //
navātapālohitam āhitam muhuḥ mahāniveśau paritaḥ payodharau /
cakāsayantīm aravindajam rajaḥ pariśramāmbhaḥpulakena sarpatā //
kapolasaṃśleṣi vilocanatviṣā vibhūṣayantīm avataṃsakotpalam /
sutena pāṇḍoḥ kalamasya gopikām nirīkṣya mene śaradaḥ kṛtārthatā //
upāratāḥ paścimarātrigocarāt apārayantaḥ patitum javena gām /
tam utsukāḥ cakruḥ avekṣaṇotsukam gavām gaṇāḥ prasnutapīvaraudharasaḥ  //
parītam ukṣāvajaye jayaśriyā nadantam uccaiḥ kṣatasindhurodhasam /
dadarśa puṣṭim dadhatam sa śāradīm savigraham darpam iva adhipam gavām //
vimucyamānaiḥ api tasya mantharam gavām himānīviśadaiḥ kadambakaiḥ  /
śarannadīnām pulinaiḥ kutūhalam galaddukūlaiḥ jaghanaiḥ iva ādadhe //
gatān paśūnām sahajanmabandhutām gṛhāśrayam prema vaneṣu bibhrataḥ  /
dadarśa gopān upadhenu pāṇḍavaḥ kṛtānukārān iva gobhiḥ ārjave //
paribhraman mūrdhajaṣaṭpadākulaiḥ smitodayādarśitadantakesaraiḥ  /
mukhaiḥ calatkuṇḍalaraśmirañjitaiḥ navātapāmṛṣṭasarojacārubhiḥ  //
nibaddhaniḥśvāsavikampitādharāḥ latāḥ iva prasphuritaikapallavāḥ  /
vyapoḍhapārśvaiḥ apavartitatrikāḥ vikarṣaṇaiḥ pāṇivihārahāribhiḥ  //
vrajājireṣu ambudanādaśaṅkinīḥ śikhaṇḍinām unmadayatsu yoṣitaḥ  /
muhuḥ praṇunneṣu mathām vivartanaiḥ nadatsu kumbheṣu mṛdaṅgamantharam //
sa mantharāvalgitapīvarastanīḥ pariśramaklāntavilocanotpalāḥ  /
nirīkṣitum na upararāma ballavīḥ abhipranṛttāḥ iva vārayoṣitaḥ  //
papāta pūrvām jahataḥ vijihmatām vṛṣopabhuktāntikasasyasampadaḥ  /
rathāṅgasīmantitasāndrakardamān prasaktasampātapṛthakkṛtān pathaḥ  //
janaiḥ upagrāmam anindyakarmabhiḥ viviktabhāveṅgitabhūṣaṇaiḥ vṛtāḥ  /
bhṛśam dadarśa āśramamaṇḍapopamāḥ sapuṣpahāsāḥ sa niveśavīrudhaḥ  //
tataḥ sa samprekṣya śaradguṇaśriyam śaradguṇālokanalolacakṣuṣam /
uvāca yakṣaḥ tam acoditaḥ api gām na hi iṅgitajñaḥ avasare avasīdati //

jayaśriyam pārtha pṛthūkarotu te śarat prasannāmbuḥ anambuvāridā //

patanti na asmin viśadāḥ patattriṇaḥ dhṛtendracāpāḥ na payodapaṅktayaḥ  /

iyam kadambānilabhartuḥ atyaye na digvadhūnām kṛśatā na rājate //

vikāsi vaprāmbhasi gandhasūcitam namanti nighrātum iva asitotpalam //
mṛṇālinīnām anurañjitam tviṣā vibhinnam ambhojapalāśaśobhayā /

vipāṇḍu saṃvyānam iva aniloddhatam nirundhatīḥ saptapalāśajam rajaḥ  /
anāvilonmīlitabāṇacakṣuṣaḥ sapuṣpahāsāḥ vanarājiyoṣitaḥ  //
adīpitam vaidyutajātavedasā sitāmbudacchedatirohitātapam /

sitacchadānām apadiśya dhāvatām rutaiḥ amīṣām grathitāḥ patatriṇām /

vihārabhūmeḥ abhighoṣam utsukāḥ śarīrajebhyaḥ cyutayūthapaṅktayaḥ  /
asaktam ūdhāṃsi payaḥ kṣaranti amūḥ upāyanāni iva nayanti dhenavaḥ  //
jagatprasūtiḥ jagadekapāvanī vrajopakaṇṭham tanayaiḥ upeyuṣī /

asau anāsthāparayā avadhīritaḥ saroruhiṇyā śirasā naman api /
upaiti śuṣyan kalamaḥ saha ambhasā manobhuvā taptaḥ iva abhipāṇḍutām //
amī samuddhūtasarojareṇunā hṛtāḥ hṛtāsārakaṇena vāyunā /
upāgame duścaritāḥ iva āpadām gatim na niścetum alam śilīmukhāḥ  //
mukhaiḥ asau vidrumabhaṅgalohitaiḥ śikhāḥ piśaṅgīḥ kalamasya bibhratī /
śukāvaliḥ vyaktaśirīṣakomalā dhanuḥśriyam gotrabhidaḥ anugacchati //

vigalitajalabhāraśuklabhāsām nicayaḥ iva ambumucām nagādhirājaḥ  //

ākīrṇam balarajasā ghanāruṇena prakṣobhaiḥ sapadi taraṅgitam taṭeṣu /
mātaṅgonmathitasarojareṇupiṅgam māñjiṣṭham vasanam iva ambu nirbabhāse //

samprāpe nisṛtamadāmbubhiḥ gajendraiḥ prasyandipracalitagaṇḍaśailaśobhā //

ātenuḥ cakitacakoranīlakaṇṭhān kacchāntān amaramahebhabṛṃhitāni //

jajñe niveśanavibhāgapariṣkṛtānām lakṣmīḥ puropavanajā vanapādapānām //

vanam viśantyaḥ vanajāyatekṣaṇāḥ kṣaṇadyutīnām dadhuḥ ekarūpatām //

iti priyam kāṃcit upaitum icchatīm puraḥ anuninye nipuṇaḥ sakhījanaḥ  //

pratīradeśaiḥ svakalatracārubhiḥ vibhūṣitāḥ kuñjasamudrayoṣitaḥ  //

priyāṅkaśītāḥ śucimauktikatviṣaḥ vanaprahāsāḥ iva vāribindavaḥ  //

sthiradvirephāñjanaśaritodaraiḥ visāribhiḥ puṣpavilocanaiḥ latāḥ  //
upeyuṣīṇām bṛhatīḥ adhityakāḥ manāṃsi jahruḥ surarājayoṣitām /

tarasā eva kaḥ api bhuvanaikapuruṣa puruṣaḥ tapasyati /
jyotiramalavapuṣaḥ api raveḥ abhibhūya vṛtraḥ iva bhīmavigrahaḥ  //
sa dhanurmaheṣudhi nibharti kavacam asitam uttamam jaṭāḥ  /
valkam ajinam iti citram idam munitāvirodhi na ca na asya rājate //
calane avaniḥ calati tasya karaṇaniyame sadiṅmukham /
stambham anubhavati śāntamarudgrahatārakāgaṇayutam nabhastalam //
sa tadojasā vijitasāram amaraditijopasaṃhitam /
viśvam idam apidadhāti purā kim iva asti yat na tapasām aduṣkaram //
vijigīṣate yadi jaganti yugapat atha saṃjihīrṣati /
prāptum abhavam abhivāñchati vā vayam asya no viṣahitum kṣamāḥ rucaḥ  //
kim upekṣase kathaya nātha na tava viditam na kiṃcana /
trātum alam abhayada arhasi naḥ tvayi mā sma śāsati bhavatparābhavaḥ  //
iti gām vidhāya virateṣu muniṣu vacanam samādade /
bhinnajaladhijalanādaguru dhvanayan diśām vivaram andhakāntakaḥ  //
badarītapovananivāsaniratam avagāta mā anyathā /
dhātuḥ udayanidhane jagatām naram aṃśam ādipuruṣasya gām gatam //
dviṣataḥ parāsisiṣuḥ eṣa sakalabhuvanābhitāpinaḥ  /
krāntakuliśakaravīryabalāt madupāsanam vihitavān mahat tapaḥ  //
ayam acyutaḥ ca vacanena sarasiruhajanmanaḥ prajāḥ  /
pātum asuranidhanena vibhū bhuvam abhyupetya manujeṣu tiṣṭhataḥ  //
surakṛtyam etat avagamya nipuṇam iti mūkadānavaḥ  /
hantum abhipatati pāṇḍusutam tvarayā tat atra saha gamyatām mayā //
vivare api na enam anigūḍham abhibhavitum eṣa pārayan /
pāpaniratiḥ aviśaṅkitayā vijayam vyavasyati varāhamāyayā //
nihate viḍambitakirātanṛpativapuṣā ripau mayā /
muktaniśitaviśikhaḥ prasabham mṛgayāvivādam ayam ācariṣyati //
tapasā nipīḍitakṛśasya virahitasahāyasampadaḥ  /
sattvavihitam atulam bhujayoḥ balam asya paśyata mṛdhe adhikupyataḥ  //
iti tān udāram anunīya viṣamaharicandanālinā /
gharmajanitapulakena lasadgajamauktikāvaliguṇena vakṣasā //
vadanena puṣpitalatāntaniyamitavilambitamaulinā /
bibhrat aruṇanayanena rucam śikhipicchalāñchitakapolabhittinā //
bṛhadudvahan jaladanādi dhanuḥ upahitaikamārgaṇam /
meghanicayaḥ iva saṃvavṛte ruciraḥ kirātapṛtanāpatiḥ śivaḥ  //
anukūlam asya ca vicintya gaṇapatibhiḥ āttavigrahaiḥ  /
śūlaparaśuśaracāpabhṛtaiḥ mahatī vanecaracamūḥ vinirmame //
viracayya kānanavibhāgam anugiram atha īśvarājñayā /
bhīmaninadapihitorubhuvaḥ paritaḥ apadiśya mṛgayām pratasthire //
kṣubhitābhiniḥsṛtavibhinnaśakunimṛgayūthaniḥsvanaiḥ  /
pūrṇapṛthuvanaguhāvivaraḥ sahasā bhayāt iva rarāsa bhūdharaḥ  //
na virodhinī ruṣam iyāya pathi mṛgavihaṅgasaṃhatiḥ  /
ghnanti sahajam api bhūribhiyaḥ samam āgatāḥ sapadi vairam āpadaḥ  //
camarīgaṇaiḥ gaṇabalasya balavati bhaye api upasthite /
vaṃśavitatiṣu viṣaktapṛthupriyabālavāladhibhiḥ ādade dhṛtiḥ  //
harasainikāḥ pratibhaye api gajamadasugandhikesaraiḥ  /
svastham abhidadṛśire sahasā pratibodhajṛmbhamukhaiḥ mṛgādhipaiḥ  //
bibharāṃbabhūvuḥ apavṛttajaṭharaśapharīkulākulāḥ  /
paṅkaviṣamitataṭāḥ saritaḥ karirugṇacandanarasāruṇam payaḥ  //
mahiṣakṣatāgurutamālanaladasurabhiḥ sadāgatiḥ  /
vyastaśukanibhaśilākusumaḥ praṇudan vavau vanasadām pariśramam //
mathitāmbhasaḥ rayavikīrṇamṛditakadalīgavedhukāḥ  /
klāntajalaruhalatāḥ sarasīḥ vidadhe nidāghaḥ iva sattvasamplavaḥ  //
iti cālayan acalasānuvanagahanajān umāpatiḥ  /
prāpa muditahariṇīdaśanakṣatavīrudham vasatim aindrasūnavīm //
sa tam āsasāda ghananīlam abhimukham upasthitam muneḥ  /
pitranikaṣaṇavibhinnabhuvam danujam dadhānam atha saukaram vapuḥ  //
kacchānte surasaritaḥ nidhāya senām anvatiḥ sakatipayaiḥ kirātavaryaiḥ  /
pracchannaḥ tarugahanaiḥ sagulmajālaiḥ lakṣmīvān anupadam asya sampratasthe //
vapuṣām parameṇa bhūdharāṇām atha sambhāvyaparākramam vibhede /
mṛgam āśu vilokayāṃcakāra sthiradaṃṣṭrogramukham mahendrasūnuḥ  //
sphuṭabaddhasaṭonnatiḥ sa dūrāt abhidhāvan avadhīritānyakṛtyaḥ  /
jayam icchati tasya jātaśaṅke manasi imam muhur ādade vitarkam //
ghanapotravidīrṇaśālamūlaḥ nibiḍaskandhanikāṣarugṇavapraḥ  /
ayam ekacaraḥ abhivartate mām samarāya iva samājuhūṣamāṇaḥ  //
iha vītabhayāḥ taponubhāvāt jahati vyālamṛgāḥ pareṣu vṛttim /
mayi tām sutarām ayam vidhatte vikṛtiḥ kim nu bhavet iyam nu māyā //
athavā eṣa kṛtajñayā iva pūrvam bhṛśam āsevitayā ruṣā na muktaḥ  /
avadhūya virodhinīḥ kim ārāt mṛgajātīḥ abhiyāti mām javena //
na mṛgaḥ khalu kaḥ api ayam jighāṃsuḥ skhalati hi atra tathā bhṛśam manaḥ me /
vimalam kaluṣībhavat ca cetaḥ kathayati eva hitaiṣiṇam ripum vā //
muniḥ asmi nirāgasaḥ kutaḥ me bhayam iti eṣa na bhūtaye abhimānaḥ  /
paravṛddhiṣu baddhamatsarāṇām kim iva hi asti durātmanām alaṅghyam //
danujaḥ svit ayam kṣapācaraḥ vā vanaje na iti balam bat asti sattve /
abhibhūya tathā hi meghanīlaḥ sakalam kampayati iva śailarājam //
ayam eva mṛgavyasattrakāmaḥ prahariṣyan mayi māyayā śamasthe /
pṛthubhiḥ dhvajinīsravaiḥ akārṣīt cakitodbhrāntamṛgāṇi kānanāni //
bahuśaḥ kṛtasatkṛteḥ vidhātum priyam icchan athavā suyodhanasya /
kṣubhitam vanagocarābhiyogāt gaṇam āśiśriyat ākulam tiraścām //
avalīḍhasanābhiḥ aśvasenaḥ prasabham khāṇḍavajātavedasā vā /
pratikartum upāgataḥ samanyuḥ kṛtamanyuḥ yadi vā vṛkodareṇa //
balaśālitayā yathā tathā vā dhiyam ucchedaparāmayam dadhānaḥ  /
niyamena mayā nibarhaṇīyaḥ paramam lābham arātibhaṅgam āhuḥ  //
kuru tāta tapāṃsi amārgadāyī vijayāya iti alam anvaśāt muniḥ mām /
balinaḥ ca vadhāt ṛte asya śakyam vrasaṃrakṣaṇam anyathā na kartum //
iti tena vicintya cāpanāma prathamam pauruṣacihnam ālalambe /
upalabdhaguṇaḥ parasya bhede sacivaḥ śuddhaḥ iva ādade ca bāṇaḥ  //
anubhāvavatā guru sthiratvāt avisaṃvādi dhanuḥ dhanaṃjayena /
svabalavyasane api pīḍyamānam guṇavat mitram iva ānatim prapede //
pravikarṣaninādabhinnarandhraḥ padaviṣṭambhanipīḍitaḥ tadānīm /
adhirohati gāṇḍivam maheṣau sakalaḥ saṃśayam āruroha śailaḥ  //
dadṛśe atha savismayam śivena sthirapūrṇāyatacāpamaṇḍalasthaḥ  /
racitaḥ tisṛṇām purām vidhātum vadham ātmā iva bhayānakaḥ pareṣām //
vicakarṣa ca saṃhiteṣuḥ uccaiḥ caraṇāskandananāmitācalendraḥ  /
dhanurāyatabhogavāsukijyāvadanagranthivimuktavahni śambhuḥ  //
sa bhavasya bhavakṣayaikahetoḥ sitasapteḥ ca vidhāsyatoḥ saha artham /
ripuḥ āpa parābhavāya madhyam prakṛtipratyayayoḥ iva anubandhaḥ  //
atha dīpitavārivāhavartmā ravavitrāsitavāraṇāt avāryaḥ  /
nipapāta javādiṣu pinākāt mahataḥ abhrāt iva vaidyutaḥ kṛśānuḥ  //
vrajataḥ asya bṛhat patattrajanmā kṛtatārkṣyopanipātavegaśaṅkaḥ  /
pratinādamahāt mahoragāṇām hṛdayaśrotrabhit utpapāta nādaḥ  //
nayanāt iva śūlinaḥ pravṛttaiḥ manasaḥ api āśutaram yataḥ piśaṅgaiḥ  /
vidadhe vilasattaḍillatābhaiḥ kiraṇaiḥ vyomani mārgaṇasya mārgaḥ  //
apayan dhanuṣaḥ śivāntikasthaiḥ vivaresadbhiḥ abhikhyayā jihānaḥ  /
yugapat dadṛśe viśan varāham tadupoḍhaiḥ ca nabhaścaraiḥ pṛṣatkaḥ  //
sa tamālanibhe ripau surāṇām ghananīhāraḥ iva aviṣaktavegaḥ  /
bhayaviplutam īkṣitaḥ nabhaḥsthaiḥ jagatīm grāhaḥ iva apagām jagāhe //
sapadi priyarūpaparvarekhaḥ sitalohāgranakhaḥ kham āsasāda /
kupitāntakatarjanāṅguliśrīḥ vyathayan prāṇabhṛtaḥ kapidhvajeṣu //
paramāstraparigrahorutejaḥ sphuradulkākṛti vikṣipan vaneṣu /
sa javena patan paraḥśatānām patatām vrātaḥ iva āravam vitene //
avibhāvitaniṣkramaprayāṇaḥ śamitāyāmaḥ iva atiraṃhasā saḥ  /
saha pūrvataram nu cittavṛtteḥ apatitvā nu cakāra lakṣyabhedam //
sa vṛṣadhvajasāyakāvabhinnam jayahetuḥ pratikāyam eṣaṇīyam /
laghu sādhayitum śaraḥ prasehe vidhinā iva artham udīritam prayatnaḥ  //
avivekavṛthāśramau iva artham kṣayalobhau iva saṃśritānurāgam /
vijigīṣum iva ānayapramādau avasādam viśikhau vininyatuḥ tam //
atha dīrghatamam tamaḥ pravekṣyan sahasā rugṇrayaḥ sa sambhrameṇa /
nipatantam iva uṣṇaraśmim urvyām valayībhūtatarum dharām ca mene //
sa gataḥ kṣitim uṣṇaśoṇitārdraḥ khuradaṃṣṭrāgranipātadāritāśmā /
asubhiḥ kṣaṇam īkṣitendrasūniḥ vihitāmarṣagurudhvaniḥ nirāse //
sphuṭapauruṣam āpapāta pārthaḥ tam atha prājyaśaraḥ śaram jighṛkṣuḥ  /
na tathā kṛtavedinām kariṣyan priyatām eti yathā kṛtāvadānaḥ  //
upakāraḥ iva asati prayuktaḥ sthitim aprāpya mṛge gataḥ praṇāśam /
kṛtaśaktiḥ avāṅmukhaḥ gurutvāt janitavrīḍaḥ iva ātmapauruṣeṇa //
sa samuddharatā vicintya tena svarucam kīrtim iva uttamām dadhānaḥ  /
anuyuktaḥ iva svavārtam uccaiḥ parirebhe nu bhṛśam vilocanābhyām //
tatra kārmukabhṛtam mahābhujaḥ paśyati sma sahasā vanecaram /
saṃnikāśayitum agrataḥ sthitam śāsanam kusumacāpavidviṣaḥ  //
sa prayujya tanaye mahīpateḥ ātmajātisadṛśīm kila ānatim /
sāntvapūrvam abhinītihetukam vaktum ittham upacakrame vacaḥ  //
śāntatā vinayayogi mānasam bhūridhāma vimalam tapaḥ śrutam /
prāha te nu sadṛśī divaukasām anvavāyam avadātam ākṛtiḥ  //
dīpitaḥ tvam anubhāvasampadā gauraveṇa laghayan mahībhṛtaḥ  /
rājase muniḥ api iha kārayan ādhipatyam iva śātamanyavam //
tāpasaḥ api vibhutām upeyivān āspadam tvam asi sarvasampadām /
dṛśyate hi bhavataḥ vinā janaiḥ anvitasya sacivaiḥ iva dyutiḥ  //
vismayaḥ kaḥ iva vā jayaśriyā na eva muktiḥ api te davīyasī /
īpsitasya na bhavet upāśrayaḥ kasya nirjitarajastamoguṇaḥ  //
hrepayan ahimatejasam tviṣā sa tvam ittham upapannapauruṣaḥ  /
hartum arhasi varāhabhedinam na enam asmadadhipasya sāyakam //
smaryate tanubhṛtām sanātanam nyāyyam ācaritam uttamaiḥ nṛbhiḥ  /
dhvaṃsate yadi bhavādṛśaḥ tataḥ kaḥ prayātu vada tena vartmanā //
ākumāram upadeṣṭum icchavaḥ saṃnivṛttim apathāt mahāpadaḥ  /
yogaśaktijitajanmamṛtyavaḥ śīlayanti yatayaḥ suśīlatām //
tiṣṭhatām tapasi puṇyam āsajan sampadaḥ anuguṇayan sukhaiṣiṇām /
yoginām pariṇaman vimuktaye kena na astu vinayaḥ satām priyaḥ  //
nūnam atrabhavataḥ śarākṛtim sarvathā ayam anuyāti sāyakaḥ  /
saḥ ayam iti anupapannasaṃśayaḥ kāritaḥ tvam apathe padam yayā //
anyadīyaviśikhe na kevalam niḥspṛhasya bhavitavyam āhṛte /
nighnataḥ paranibarhitam mṛgam vrīḍitavyam api te sacetasaḥ  //
saṃtatam niśamayantaḥ utsukāḥ yaiḥ prayānti mudam asya sūrayaḥ  /
kīrtitāni hasite api tāni yam vrīḍayanti caritāni māninam //
anyadoṣam iva saḥ svakam guṇam khyāpayet katham adhṛṣṭatājaḍaḥ  /
ucyate sa khalu kāryavattayā dhik vibhinnabudhasetum arthitām //
durvacam tat atha mā sma bhūt mṛgaḥ tvau asau yat akariṣyat ojasā /
na enam āśu yadi vāhinīpatiḥ pratyapatsyata śitena pattriṇā //
kaḥ nu imam harituraṅgam āyudhastheyasīm dadhatam aṅgasaṃhatim /
vegavattaramṛte camūpateḥ hantum arhati śareṇa daṃṣṭriṇam //
mitram iṣṭam upakāri saṃśaye medinīpatiḥ ayam tathā ca te /
tam virodhya bhavatā nirāsi mā sajjanaikavasatiḥ kṛtajñatā //
labhyam ekasukṛtena durlabhāḥ rakṣitāram asurakṣyabhūtayaḥ  /
svantam antavirasāḥ jigīṣatām mitralābham anu lābhasampadaḥ  //
cañcalam vasu nitāntam unnatāḥ medinīm api haranti arātayaḥ  /
bhūdharasthiram upeyam āgatam mā avamaṃsta suhṛdam mahīpatim //
jetum eva bhavatā tapasyate na āyudhāni dadhate mumukṣavaḥ  /
prāpsyate ca sakalam mahībhṛtā saṃgatena tapasaḥ phalam tvayā //
vājibhūmiḥ ibharājakānanam santi ratnanicayāḥ ca bhūriśaḥ  /
kāñcanena kim iva asya pattriṇā kevalam na sahate vilaṅghanam //
sāvalepam upalipsate paraiḥ abhyupaiti vikṛtim rajasi api /
arthitaḥ tu na mahān samīhate jīvitam kimu dhanam dhanāyitum //
tat tadīyaviśikhātisarjanāt astu vām guru yadṛcchayā āgatam /
rāghavaplavagarājayoḥ iva prema yuktam itaretarāśrayam //
na abhiyoktum anṛtam tvam iṣyate kaḥ tapasviviśikheṣu ca ādaraḥ  /
santi bhūbhṛti śarāḥ hi naḥ pare ye parākramavasūni vajriṇaḥ  //
mārgaṇaiḥ atha tava prayojanam nāthase kimu patim na bhūbhṛtaḥ  /
tvadvidham suhṛdam etya saḥ arthinam kim na yacchati vijitya medinīm //
tena sūriḥ upakāritādhanaḥ kartum icchati na yācitam vṛthā /
sīdatām anubhavan iva arthinām veda yat praṇayabhaṅgavedanām //
śaktiḥ arthapatiṣu svayaṃgraham prema kārayati vā niratyayam /
kāraṇadvayam idam nirasyataḥ prārthanādhikabale vipatphalā //
astravedam adhigamya tattvataḥ kasya ca iha bhujavīryaśālinaḥ  /
jāmadagnyam apahāya gīyate tāpaseṣu caritārtham āyudham //
abhyaghāni municāpalāt tvayā yat mṛgaḥ kṣitipateḥ parigrahaḥ  /
akṣamiṣṭa tat ayam pramādyatām saṃvṛṇoti khalu doṣam ajñatā //
janmaveṣatapasām virodhinīm mā kṛthāḥ punar amūm apakriyām /
āpat eti ubhayalokadūṣaṇī vartamānam apathe hi durmatim //
yaṣṭum icchasi pitṝn na sāmpratam saṃvṛtaḥ arcicayiṣuḥ divaukasaḥ  /
dātum eva padavīm api kṣamaḥ kim mṛge aṅga viśikham nyavīviśaḥ  //
sajjanaḥ asi vijahīhi cāpalam sarvadā kaḥ iva vā sahiṣyate /
vāridhīn iva yugāntavāyavaḥ kṣobhayanti anibhṛtāḥ gurūn api //
astravedavit ayam mahīpatiḥ parvatīyaḥ iti mā avajīgaṇaḥ  /
gopitum bhuvam imām marutvatā śailavāsam anunīya lambhitaḥ  //
tat titikṣitam idam mayā muneḥ iti avocata vacaḥ camūpatiḥ  /
bāṇam atrabhavate nijam diśan āpnuhi tvam api sarvasampadaḥ  //
ātmanīnam upatiṣṭhate guṇāḥ sambhavanti viramanti ca āpadaḥ  /
iti anekaphalabhāji mā sma bhūt arthitā katham iva āryasaṃgame //
dṛśyatām ayam anokahāntare tigmahetipṛtanābhiḥ anvitaḥ  /
sāhivīciḥ iva sindhuḥ uddhataḥ bhūpatiḥ samayasetuvāritaḥ  //
sajyam dhanuḥ vahati yaḥ ahipatisthavīyaḥ stheyān jayan harituraṅgamaketulakṣmīm /
asya anukūlaya matim matiman anena sakhyā sukham samabhiyāsyasi cintitāni //
tataḥ kirātasya vacobhiḥ uddhataiḥ parāhataḥ śailaḥ iva arṇavāmbubhiḥ  /
jahau na dhairyam kupitaḥ api pāṇḍavaḥ sudurgrahāntaḥkaraṇāḥ hi sādhavaḥ  //
saleśam ulliṅgitaśātraveṅgitaḥ kṛtī girām vistaratattvasaṃgrahe /
ayam pramāṇīkṛtakālasādhanaḥ praśāntasaṃrambhaḥ iva ādade vacaḥ  //
viviktavarṇābharaṇā sukhaśrutiḥ prasādayantī hṛdayāni api dviṣām /
pravartate na akṛtapuṇyakarmaṇām prasannagambhīrapadā sarasvatī //
bhavanti te sabhyatamāḥ vipaścitām manogatam vāci niveśayanti ye /
nayanti teṣu api upapannanaipuṇāḥ gambhīram artham katicit prakāśatām //
stuvanti gurvīm abhidheyasampadam viśuddhimukteḥ apare vipaścitaḥ  /
iti sthitāyām pratipūruṣam rucau sudurlabhāḥ sarvamanoramāḥ giraḥ  //
samasya sampādayatā guṇaiḥ imām tvayā samāropitabhāra bhāratīm /
pragalbham ātmā dhuri dhurya vāgminām vanacareṇa api satā adhiropitaḥ  //
prayujya sāmācaritam vilobhanam bhayam vibhedāya dhiyaḥ pradarśitam /
tathā abhiyuktam ca śilīmukhārthinā yathā itarat nyāyyam iva avabhāsate //
virodhi siddheḥ iti kartum udyataḥ sa vāritaḥ kim bhavatā na bhūpatiḥ  /
hite niyojyaḥ khalu bhūtim icchatā saha arthanāśena nṛpaḥ anujīvinā //
dhruvam praṇāśaḥ prahitasya pattriṇaḥ śiloccaye tasya vimārgaṇam nayaḥ  /
na yuktam atra āryajanātilaṅghanam diśati apāyam hi satām atikramaḥ  //

parāṅmukhatve api kṛtopakārāt tūṇīmukhāt mitrakulāt iva āryaḥ  //
paścātkriyā tūṇayugasya bhartuḥ jajñe tadānīm upakāriṇī iva /
sambhāvanāyām adharīkṛtāyām patyuḥ puraḥ sāhasam āsitavyam //
tam śambhuḥ ākṣiptamaheṣujālam lohaiḥ śaraiḥ marmasu nistutoda /
hṛttottaram tattvavicāramadhye vaktā iva doṣaiḥ gurubhiḥ vipakṣam //
jahāra ca asmāt acireṇa varma jvalanmaṇidyotitahaimalekham /
caṇḍaḥ pataṅgāt marudekanīlam taḍitvataḥ khaṇḍam iva ambudasya //
vikośanirdhautatanoḥ mahāseḥ phaṇāvataḥ ca tvaci vicyutāyām /
pratidvipābaddharuṣaḥ samakṣam nāgasya ca ākṣiptamukhacchadasya //
vibodhitasya dhvaninā ghanānām hareḥ apetasya ca śailarandhrāt /
nirastadhūmasya ca rātrivahneḥ vinā tanutreṇa rucim sa bheje //
acittatāyām api nāma yuktām anūrdhvatām prāpya tadīyakṛcchre /
mahīm gatau tau iṣudhī tadānīm vivavratuḥ cetanayā iva yogam //
sthitam viśuddhe nabhasi iva sattve dhāmnā tapovīryamayena yuktam /
śastrābhighātaiḥ tam ajasram īśaḥ tvaṣṭā vivasvantam iva ullilekha //
saṃrambhavegojjhitavedaneṣu gātreṣu bāhiryam upāgateṣu /
muneḥ babhūva agaṇiteṣurāśeḥ lauhaḥ tiraskāraḥ iva ātmamanyuḥ  //
tataḥ anupūrvāyatavṛttabāhuḥ śrīmān kṣarallohitadigdhadehaḥ  /
āskandya vegena vimuktanādaḥ kṣitim vidhunvan iva pārṣṇighātaiḥ  //
sāmyam gatena aśaninā maghonaḥ śaśāṅkakhaṇḍākṛtipāṇḍureṇa /
śambhum bibhitsuḥ dhanuṣā jaghāna stambam viṣāṇena mahān iva ibhaḥ  //
rayeṇa sā saṃnidadhe patantī bhavodbhavena ātmani cāpayaṣṭiḥ  /
samuddhatā sindhuḥ anekamārgā pare sthitena ojasi jahnunā iva //
vikārmukaḥ karmasu śocanīyaḥ paricyutaudāryaḥ iva upacāraḥ  /
vicikṣipe śūlabhṛtā salīlam sa patribhiḥ dūram adūrapātaiḥ  //
upoḍhakalyāṇaphalaḥ abhirakṣan vīravratam puṇyaraṇāśramasthaḥ  /
japopavāsaiḥ iva saṃyatātmā tepe muniḥ taiḥ iṣubhiḥ śivasya //
tataḥ agrabhūmim vyavasāyasiddheḥ sīmānam anyaiḥ atidustaram saḥ  /
tejaḥśriyām āśrayam uttamāsim sākṣāt ahaṃkāram iva ālalambe //
śarān avadyan anavadyakarmā cacāra citram pravicāramārgaiḥ  /
hastena nistriṃśabhṛtā sa dīptaḥ sārkāṃśunā vāridhiḥ ūrmiṇā iva //
yathā nije vartmani bhāti bhābhiḥ cyāyāmayaḥ ca apsu sahasraraśmiḥ  /
tathā nabhasi āśu raṇasthalīṣu spaṣṭadvimūrtiḥ dadṛśe sa bhūtaiḥ  //
śivapraṇunnena śilīmukhena tsarupradeśāt apavarjitāṅgaḥ  /
jvalan asiḥ tasya papāta pāṇeḥ ghanasya vaprāt iva vaidyutaḥ agniḥ  //
ākṣiptacāpāvaraṇeṣujālaḥ chinnottamāsiḥ sa mṛdhe avadhūtaḥ  /
riktaḥ prakāśaḥ ca babhūva bhūmeḥ utsāditodyānaḥ iva pradeśaḥ  //
sa khaṇḍanam prāpya parāt amarṣavān bhujadvitīyaḥ api vijetum icchayā /
sasarja vṛṣṭim parirugṇapādapām dravetareṣām payasām iva aśmanām //
nīrandhram parigamite kṣayam pṛṣatkaiḥ bhūtānām adhipatinā śilāvitāne /
ucchrāyasthagitanabhodigantarālam cikṣepa kṣitiruhajālam indrasūnuḥ  //
niḥśeṣam śakalitavalkalāṅgasāraiḥ kurvadbhiḥ bhuvam abhitaḥ kaṣāyacitrām /
īśānaḥ sakusumapallavaiḥ nagaiḥ taiḥ ātene balim iva raṅgadevatābhyaḥ  //
unmajjan makaraḥ iva amārāpagāyāḥ vegena pratimukham etya bāṇanadyāḥ  /
gāṇḍīvī kanakaśilānibham bhujābhyām ājaghne viṣam avilocanasya vakṣaḥ  //
abhilaṣataḥ upāyam vikramam kīrtilakṣmyoḥ asugamam arisainyaiḥ aṅkam abhyāgatasya /
janakaḥ iva śiśutve supriyasya ekasūnoḥ avinayam api sehe pāṇḍavasya smarāriḥ  //
tataḥ udagre iva dvirade munau raṇam upeyuṣi bhīmabhujāyudhe /
dhanuḥ apāsya sabāṇadhi śaṃkaraḥ pratijaghāna ghanaiḥ iva muṣṭibhiḥ  //
harapṛthāsutayoḥ dhvaniḥ utpatan amṛdusaṃvalitāṅgulipāṇijaḥ  /
sphuṭadanalpaśilāravadāruṇaḥ pratinanāda darīṣu darībhṛtaḥ  //
śivabhujāhatibhinnapṛthukṣatīḥ sukham iva anubabhūva kapidhvajaḥ  /
kaḥ iva nāma bṛhanmanasām bhavet anukṛteḥ api sattvavatām kṣamaḥ  //
vraṇamukhacyutaśoṇitaśīkarasthagitaśailataṭābhabhujāntaraḥ  /
abhinavauṣasarāgabhṛtā babhau jaladhareṇa samānam umāpatiḥ  //
urasi śūlabhṛtaḥ prahitā muhuḥ pratihatim yayuḥ arjunamuṣṭayaḥ  /
bhṛśarayāḥ iva sahyamahībhṛtaḥ pṛthuni rodhasi sindhumahormayaḥ  //
nipatite adhiśirodharam āyate samam aratniyuge ayugacakṣuṣaḥ  /
tricatureṣu padeṣu kirīṭinā lulitadṛṣṭi madāt iva caskhale //
abhibhavoditamanyuvidīpitaḥ samabhisṛtya bhṛśam javam ojasā /
bhujayugena vibhajya samādade śaśikalābharaṇasya bhujadvayam //
pravavṛte atha mahāhavamallayoḥ acalasaṃcalanāharaṇaḥ raṇaḥ  /
karaṇaśṛṅkhalasaṃkalanāguruḥ gurubhujāyudhagarvitayoḥ tayoḥ  //
ayam asau bhagavān uta pāṇḍavaḥ sthitam avāk muninā śaśimaulinā /
samadhirūḍham ajena nu jiṣṇunā svit iti vegavaśāt mumuhe gaṇaiḥ  //
pracalite calitam sthitam āsthite vinamite natam unnatam unnatau /
vṛṣakapidhvajayoḥ asahiṣṇunā muhuḥ abhāvabhayāt iva bhūbhṛtā //
karaṇaśṛṅkhalaniḥsṛtayoḥ tayoḥ kṛtabhujadhvani valgu vivalgatoḥ  /
caraṇapātanipātitarodharasaḥ prasasṛpuḥ saritaḥ paritaḥ sthalīḥ  //
viyati vegapariplutam antarā samabhisṛtya rayeṇa kapidhvajaḥ  /
caraṇayoḥ caraṇānamitakṣitiḥ nijagṛhe tisṛṇām jayinam purām //
vismitaḥ sapadi tena karmaṇā karmaṇām kṣayakaraḥ paraḥ pumān /
kṣeptukāmam avanau tam aklamam niṣpipeṣa parirabhya vakṣasā //
tapasā tathā na mudam asya yayau bhagavān yathā vipulasattvatayā /
guṇasaṃhateḥ samatiriktam aho nijam eva sattvam upakāri satām //
atha himaśucibhasmabhūṣitam śirasi virājitam indulekhayā /
svavapuḥ atimanoharam haram dadhatam udīkṣya nanāma pāṇḍavaḥ  //
sahaśaradhi nijam tathā kārmukam vapuḥ atanu tathā eva saṃvarmitam /
nihitam api tathā eva paśyan asim vṛṣabhagatiḥ upāyayau vismayam //
siṣicuḥ avanim ambuvāhāḥ śanaiḥ surakusumam iyāya citram divaḥ  /
vimalaruci bhṛśam nabhaḥ dundubheḥ dhvaniḥ akhilam anāhatasya ānaśe //
āseduṣām gotrabhidaḥ anuvṛttyā gopāyakānām bhuvanatrayasya /
rociṣṇuratnāvalibhiḥ vimānaiḥ dyauḥ ācitā tārakitā iva reje //
haṃsāḥ bṛhantaḥ surasadmavāhāḥ saṃhrādikaṇṭhābharaṇāḥ patantaḥ  /
cakruḥ prayatnena vikīryamāṇaiḥ vyomnaḥ pariṣvaṅgam iva agrapakṣaiḥ  //
muditamadhulihaḥ vitānīkṛtāḥ srajaḥ upari vitatya sātānikīḥ  /
jaladaḥ iva niṣedivāṃsam vṛṣe marudupasukhayāṃbabhūva īśvaram //
kṛtadhṛti parivanditena uccakaiḥ gaṇapatibhiḥ abhinnaromodgamaiḥ  /
tapasi kṛtaphale phalajyāyasī stutiḥ iti jagade hareḥ sūnunā //
śaraṇam bhavantam atikāruṇikam bhava bhaktigamyam adhigamya janāḥ  /
jitamṛtyavaḥ ajita bhavanti bhaye sasurāsurasya jagataḥ śaraṇam //
vipat eti tāvat avasādakarī na ca kāmasampat abhikāmayate /
na namanti ca ekapuruṣam puruṣāḥ tava yāvat īśa na natiḥ kriyate //
saṃsevante dānaśīlāḥ vimuktya sampaśyantaḥ janmaduḥkham pumāṃsaḥ  /
yanniḥsaṅgaḥ tvam phalasya ānatebhyaḥ tat kāruṇyam kevalam na svakāryam //
prāpyate yat iha dūram agatvā yat phalati aparalokagatāya /
tīrtham asti na bhavārṇavabāhyam sārvakāmikam ṛte bhavataḥ tat //
vrajati śuci padam tu ati prītimān pratihatam atiḥ eti ghorām gatim /
iyam anagha nimittaśaktiḥ parā tava varada na cittabhedaḥ kvacit //
dakṣiṇām praṇatadakṣiṇa mūrtim tattvataḥ śivakarīm aviditvā /
rāgiṇā api vihitā tava bhaktyā saṃsmṛtiḥ bhava bhavati abhavāya //
dṛṣṭvā dṛśyāni ācaraṇīyāni vidhāya prekṣākārī yāti padam muktam apāyaiḥ  /
samyagdṛṣṭiḥ tasya param paśyati yaḥ tvām yaḥ ca upāsti sādhu vidheyam sa vidhatte //
yuktāḥ svaśaktyā munayaḥ prajānām hitopadeśaiḥ upakāravantaḥ  /
samucchinatsi tvam acintyadhāmā karmāṇi upetasya duruttarāṇi //
saṃnibaddham apahartum ahāryam bhūri durgatibhayam bhuvanānām /
adbhutākṛtim imām atimāyaḥ tvam bibharṣi karuṇāmaya māyām //
na rāgi cetaḥ paramā vilāsitā vadhūḥ śarīre asti na ca asti manmathaḥ  /
namaskriyā ca uṣasi dātuḥ iti aho nisargadurbodham idam tava īhitam //
tava uttarīyam karicarma sāṅgajam jvalanmaṇiḥ sāraśanam mahānahiḥ  /
srak āsyapaṅktiḥ śavabhasma candanam kalā himāṃśoḥ ca samam cakāsati //
avigrahasya api atulena hetunā sametabhinnadvayamūrti tiṣṭhataḥ  /
tava eva na anyasya jagatsu dṛśyate viruddhaveṣābharaṇasya kāntatā //
ātmalābhapariṇāmanirodhaiḥ bhūtasaṃghaḥ iva na tvam upetaḥ  /
tena sarvabhuvanātiga loke na upamānam asi na api upemayaḥ  //
tvam antakaḥ sthāvarajaṅgamānām tvayā jagat prāṇiti deva viśvam /
tvam yoginām hetuphale ruṇatsi tvam kāraṇam kāraṇakāraṇānām //
rakṣobhiḥ suramanujaiḥ diteḥ sutaiḥ vā yat lokeṣu avikalam āptam ādhipatyam /
pāvinyāḥ śaraṇagatārtihāriṇe tat māhātmyam bhava bhavate namaskriyāyāḥ  //
tarasā bhuvanāni yaḥ bibharti dhvanati brahma yataḥ param pavitram /
paritaḥ duritāni yaḥ punīte śiva tasmai pavanātane namaḥ te //
bhavataḥ smaratām sadāsane jayini brahmamaye niṣeduṣām /
dahate bhavabījasaṃtatim śikhine anekaśikhāya te namaḥ  //
ābādhāmaraṇabhayārciṣā cirāya pluṣṭebhyaḥ bhava mahatā bhavānalena /
nirvāṇam samupagamena yacchate te bījānām prabhava namaḥ astu jīvanāya //
yaḥ sarveṣām āvarītā varīyān sarvaiḥ bhāvaiḥ na āvṛtaḥ anādiniṣṭhaḥ  /
mārgātītāya indriyāṇām namaḥ te avijñeyāya vyomarūpāya tasmai //
aṇīyase viśvavidhāriṇe namaḥ namaḥ antikasthāya namaḥ davīyase /
atītya vācām manasām ca gocaram sthitāya te tatpataye namaḥ namaḥ  //
asaṃvidānasya mama īśa saṃvidām titikṣitum duścaritam tvam arhasi /
virodhya mohāt punar abhyupeyuṣām gatiḥ bhavān eva durātmanā api //
āstikyaśuddham avataḥ priyadharma dharmam dharmātmajasya vihitāgasi śatruvarge /
samprāpnuyām vijayam īśa yayā samṛddhyā tām bhūtanātha vibhutām vitara āhaveṣu //
iti nigaditavantam sūnum uccaiḥ maghonaḥ praṇataśirasam īśaḥ sādaram sāntvayitvā /
jvaladanalaparītam raudram astram dadhānam dhanurupapadam asmai vedam abhyādideśa //
sa piṅgākṣaḥ śrīmān bhuvanamahanīyena mahasā tanum bhīmām bibhrat triguṇaparivārapraharaṇaḥ  /
parītya īśānam triḥ stutibhiḥ upagītaḥ suragaṇaiḥ sutam pāṇḍoḥ vīram jaladam iva bhāsvān abhiyayau //
atha śaśadharamauleḥ abhyanujñām avāpya tridaśapatipurogāḥ pūrṇakāmāya tasmai /
avitathaphalam āśirvādam āropayantaḥ vijayi vividham astram lokapālāḥ viteruḥ  //
asaṃhāryotsāham jayinam udayam prāpya tarasā dhuram gurvīm voḍhum sthitam anavasādāya jagataḥ  /
svadhāmnā lokānām tam upari kṛtasthānam amarāḥ tapolakṣmyā dīptam dinakṛtam iva uccaiḥ upajaguḥ  //
vraja jaya ripulokam pādapadmānataḥ san gaditaḥ iti śivena ślāghitaḥ devasaṃghaiḥ  /
nijagṛham atha gatvā sādaram pāṇḍuputraḥ dhṛtagurujayalakṣmīḥ dharmasūnum nanāma //
